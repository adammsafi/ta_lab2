# API Reference

This directory contains API reference documentation for ta_lab2 components.

## Available Documentation

- **[Memory API](memory.md)** - REST endpoints for semantic memory search and health monitoring
- **[Orchestrator CLI](orchestrator.md)** - Command-line interface for AI task coordination

## Overview

The ta_lab2 API provides two main interfaces:

1. **Memory API**: FastAPI REST endpoints for cross-platform memory access (`/api/v1/memory/*`)
2. **Orchestrator CLI**: Command-line tools for multi-platform AI coordination (`ta-lab2 orchestrator`)

Both interfaces are designed for programmatic access and can be integrated into automated workflows.
