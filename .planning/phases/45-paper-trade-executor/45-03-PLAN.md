Class `SignalReader`:
- `__init__(self, engine)`: store engine reference

- `check_signal_freshness(self, conn, signal_table, signal_id, cadence_hours, last_watermark_ts)`:
  - If `last_watermark_ts is None`: skip check (first run), return silently
  - Query `SELECT MAX(ts) AS latest_ts FROM public.{signal_table} WHERE signal_id = :signal_id`
  - CRITICAL: Validate signal_table against SIGNAL_TABLE_MAP.values() before interpolation (prevent SQL injection)
  - If no signals found: raise StaleSignalError
  - Compute age_hours = (datetime.now(UTC) - latest_ts).total_seconds() / 3600
  - If age_hours > cadence_hours: raise StaleSignalError with descriptive message

- `read_unprocessed_signals(self, conn, signal_table, signal_id, last_watermark_ts) -> list[dict]`:
  - CRITICAL: Validate signal_table against SIGNAL_TABLE_MAP.values()
  - Build query: SELECT id, ts, signal_id, direction, position_state, entry_price, entry_ts, exit_price, exit_ts, feature_snapshot, params_hash FROM {signal_table} WHERE signal_id = :signal_id AND executor_processed_at IS NULL
  - If last_watermark_ts is not None: add AND ts > :watermark_ts
  - ORDER BY ts ASC
  - Return list of dicts via row._mapping

- `mark_signals_processed(self, conn, signal_table, signal_ids_and_timestamps)`:
  - For each (id, ts) pair: UPDATE {signal_table} SET executor_processed_at = now() WHERE id = :id AND ts = :ts AND signal_id = :signal_id
  - Validates signal_table before interpolation

- `get_latest_signal_per_asset(self, signals: list[dict]) -> dict[int, dict]`:
  - Pure function (no DB). Groups by id (asset_id), returns the latest (max ts) signal per asset.
  - Returns dict mapping asset_id -> signal dict

REGIME_MULTIPLIERS = {
    "bull_low_vol":  Decimal("1.0"),
    "bull_high_vol": Decimal("0.7"),
    "ranging":       Decimal("0.5"),
    "bear_low_vol":  Decimal("0.3"),
    "bear_high_vol": Decimal("0.0"),
}

Class `PositionSizer` (stateless, all static methods):

- `compute_target_position(latest_signal, portfolio_value, current_price, config) -> Decimal`:
  - If latest_signal is None or position_state == "closed": return Decimal("0")
  - direction = latest_signal["direction"]
  - fraction = Decimal(str(config.position_fraction))
  - If sizing_mode == "regime_adjusted": fraction *= REGIME_MULTIPLIERS.get(regime_label, Decimal("1.0"))
  - If sizing_mode == "signal_strength": fraction *= max(Decimal(str(signal_confidence)), Decimal("0.10"))
  - Clamp fraction at max_position_fraction
  - qty = (portfolio_value * fraction) / current_price
  - Return qty if direction == "long" else -qty

- `compute_order_delta(current_qty: Decimal, target_qty: Decimal) -> Decimal`:
  - return target_qty - current_qty

- `get_portfolio_value(conn, strategy_id, fallback_engine=None) -> Decimal`:
  - Query v_cmc_positions_agg for total position value
  - Start with initial_capital from config (e.g., Decimal("100000") default)
  - Add realized_pnl + unrealized_pnl from positions
  - If no positions exist, return initial_capital
  - Log which price source used (bar close vs exchange mid)

- `get_current_price(conn, asset_id) -> Decimal`:
  - Try exchange_price_feed first (SELECT last_price WHERE asset_id = :id ORDER BY fetched_at DESC LIMIT 1)
  - If stale (>24h old) or missing, fall back to cmc_price_bars_multi_tf (SELECT close WHERE id = :id AND tf = '1D' ORDER BY ts DESC LIMIT 1)
  - Return Decimal(str(price))
  - Log which source was used
