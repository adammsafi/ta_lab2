**ParityChecker class:**

```python
class ParityChecker:
    """
    Compare paper executor replay fills against stored backtest trades.

    Usage:
    1. Run executor in replay mode: --replay-historical --start X --end Y
    2. Run parity check: ParityChecker(engine).check(config, start, end)
    3. Returns parity report dict with pass/fail
    """
```

**check(self, config_id, signal_id, start_date, end_date, slippage_mode) -> dict:**

1. Load backtest trades:
```sql
SELECT bt.entry_ts, bt.exit_ts, bt.entry_price, bt.exit_price,
       bt.direction, bt.pnl, bt.pnl_pct
FROM cmc_backtest_trades bt
JOIN cmc_backtest_runs br ON bt.run_id = br.run_id
WHERE br.signal_id = :signal_id
  AND bt.entry_ts >= :start_date
  AND bt.entry_ts <= :end_date
ORDER BY bt.entry_ts ASC
```

2. Load executor fills:
```sql
SELECT f.filled_at, f.fill_price, f.fill_qty, f.side,
       o.asset_id, o.signal_id
FROM cmc_fills f
JOIN cmc_orders o ON f.order_id = o.order_id
WHERE o.signal_id = :signal_id
  AND f.filled_at >= :start_date
  AND f.filled_at <= :end_date
ORDER BY f.filled_at ASC
```

NOTE: The executor stores signal_id on cmc_orders (from the Phase 44 schema). If the column mapping differs at execution time, adapt the query to match actual schema.

3. Build parity report:
```python
report = {
    "config_id": config_id,
    "signal_id": signal_id,
    "date_range": f"{start_date} to {end_date}",
    "slippage_mode": slippage_mode,
    "backtest_trade_count": len(bt_trades),
    "executor_fill_count": len(exec_fills),
    "trade_count_match": len(bt_trades) == len(exec_fills),
    "max_price_divergence_bps": None,
    "pnl_correlation": None,
    "tracking_error_pct": None,
    "parity_pass": False,
}
```

4. If trade_count_match:
   - Compare fill prices pairwise (aligned by timestamp):
     - For each pair: divergence_bps = abs(exec_fill_price - bt_entry_price) / bt_entry_price * 10000
     - report["max_price_divergence_bps"] = max(divergences)
   - Compute daily P&L correlation using numpy:
     - Build daily P&L series from both trade lists
     - report["pnl_correlation"] = numpy.corrcoef(bt_pnl, exec_pnl)[0, 1]
     - report["tracking_error_pct"] = numpy.std(bt_pnl - exec_pnl) / numpy.std(bt_pnl) * 100

5. Apply parity tolerance:
   - If slippage_mode == "zero":
     - parity_pass = trade_count_match AND max_price_divergence_bps < 1.0
   - If slippage_mode in ("fixed", "lognormal"):
     - parity_pass = pnl_correlation is not None AND pnl_correlation >= 0.99

6. Return report

**format_report(self, report: dict) -> str:**
- Human-readable formatted output:
```
=== BACKTEST PARITY REPORT ===
Signal ID:        {signal_id}
Date Range:       {date_range}
Slippage Mode:    {slippage_mode}

Backtest Trades:  {backtest_trade_count}
Executor Fills:   {executor_fill_count}
Trade Count:      {'MATCH' if trade_count_match else 'MISMATCH'}

Max Price Div:    {max_price_divergence_bps:.2f} bps
P&L Correlation:  {pnl_correlation:.4f}
Tracking Error:   {tracking_error_pct:.2f}%

RESULT:           {'PASS' if parity_pass else 'FAIL'}
```

**run_parity_check.py:**
```python
#!/usr/bin/env python
"""
Backtest parity verification report.

Compares paper executor replay fills against stored backtest results.
Prerequisite: Run executor in replay mode first:
    python -m ta_lab2.scripts.executor.run_paper_executor --replay-historical --start 2024-01-01 --end 2025-01-01

Usage:
    python -m ta_lab2.scripts.executor.run_parity_check --signal-id 1 --start 2024-01-01 --end 2025-01-01
    python -m ta_lab2.scripts.executor.run_parity_check --signal-id 1 --start 2024-01-01 --end 2025-01-01 --verbose
"""
```

Arguments:
- --signal-id: Signal ID to compare (required)
- --config-id: Executor config ID (optional, for logging)
- --start: Start date (required)
- --end: End date (required)
- --slippage-mode: Expected slippage mode (default: "zero")
- --db-url: Database URL
- --verbose: Show per-trade comparison detail

Main logic:
1. Create engine with NullPool
2. Instantiate ParityChecker(engine)
3. Call check(config_id, signal_id, start, end, slippage_mode)
4. Print formatted report
5. If --verbose: print per-trade price comparison table
6. Exit 0 if parity_pass, 1 if not
