**PaperExecutor class -- run loop and config loading:**

```python
class PaperExecutor:
    """
    Reads signals, generates paper orders, simulates fills, updates positions.

    Per strategy (from dim_executor_config where is_active=TRUE):
    1. Load config from DB
    2. Check signal freshness (stale guard)
    3. Read unprocessed signals via watermark + status flag
    4. For each asset with signals: compute target vs current position
    5. Generate CanonicalOrder for delta (with signal_id set)
    6. Log to paper_orders via PaperOrderLogger
    7. Promote to cmc_orders via OrderManager.promote_paper_order
    8. Simulate fill via FillSimulator
    9. Process fill via OrderManager.process_fill (with strategy_id)
    10. Mark signals as processed, update watermark
    11. Write run log entry
    """

    def __init__(self, engine: Engine):
        self.engine = engine
        self.signal_reader = SignalReader(engine)
        self.logger = logging.getLogger(__name__)

    def run(self, dry_run: bool = False, replay_historical: bool = False,
            replay_start: str | None = None, replay_end: str | None = None) -> dict:
        """Execute paper trading for all active strategies. Returns run summary."""
        configs = self._load_active_configs()
        if not configs:
            self.logger.info("No active executor configs found")
            return {"status": "no_configs", "strategies_processed": 0}

        summary = {
            "status": "success",
            "strategies_processed": 0,
            "total_signals": 0,
            "total_orders": 0,
            "total_fills": 0,
            "total_skipped": 0,
            "errors": [],
        }

        for config in configs:
            try:
                result = self._run_strategy(config, dry_run=dry_run,
                                           replay_historical=replay_historical,
                                           replay_start=replay_start,
                                           replay_end=replay_end)
                summary["strategies_processed"] += 1
                summary["total_signals"] += result.get("signals_read", 0)
                summary["total_orders"] += result.get("orders_generated", 0)
                summary["total_fills"] += result.get("fills_processed", 0)
                summary["total_skipped"] += result.get("skipped_no_delta", 0)
            except StaleSignalError as e:
                self.logger.error("STALE SIGNAL for %s: %s", config.config_name, e)
                self._try_telegram_alert(f"STALE SIGNAL for {config.config_name}: {e}")
                self._write_run_log(config, status="stale_signal", error=str(e))
                summary["errors"].append({"config": config.config_name, "error": str(e)})
            except Exception as e:
                self.logger.exception("Executor failed for %s", config.config_name)
                self._write_run_log(config, status="failed", error=str(e))
                summary["errors"].append({"config": config.config_name, "error": str(e)})
                summary["status"] = "partial_failure"

        if summary["errors"] and summary["strategies_processed"] == 0:
            summary["status"] = "failed"

        return summary
```

**_load_active_configs(self) -> list[ExecutorConfig]:**
- Query: SELECT * FROM dim_executor_config WHERE is_active = TRUE ORDER BY config_id
- Map each row to ExecutorConfig dataclass
- Return list

**_run_strategy(self, config, dry_run, replay_historical, replay_start, replay_end) -> dict:**
- Determine signal_table from SIGNAL_TABLE_MAP[config.signal_type]
- Open connection: with self.engine.connect() as conn:
  - Step 1: check_signal_freshness (skip if replay_historical or first run)
  - Step 2: read_unprocessed_signals (or read_historical_signals if replay mode)
  - If no signals: log and return {signals_read: 0}
  - Step 3: get_latest_signal_per_asset
- For each (asset_id, signal) pair:
  - Call _process_asset_signal(asset_id, signal, config, dry_run)
- Update watermark: UPDATE dim_executor_config SET last_processed_signal_ts = :max_ts WHERE config_id = :id
- Mark signals as processed
- Write run log entry
- Return counts dict

**_write_run_log(self, config, status, signals_read=0, orders=0, fills=0, skipped=0, error=None):**
- INSERT INTO cmc_executor_run_log (config_ids, status, signals_read, orders_generated, fills_processed, skipped_no_delta, error_message, finished_at)

**_try_telegram_alert(self, message):**
- Try importing send_critical_alert from project alerting module
- If import fails or sending fails, log warning and continue
- NEVER let alerting failure crash the executor
