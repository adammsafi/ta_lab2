**Public method:** `generate_report(self, start_date: date, end_date: date, output_dir: str = "reports/validation") -> str`

Returns path to generated `V1_VALIDATION_REPORT.md`.

**Implementation:**

1. Create output_dir and charts subdir (`output_dir/charts/`).

2. Run `build_gate_scorecard(self._engine, start_date, end_date)` to get all gates.

3. Generate 5 charts (each returns relative path string):

   **Chart 1: Equity curve overlay** (`_build_equity_curve`)
   - Query `cmc_fills f JOIN cmc_orders o ON f.order_id = o.order_id` for date range
   - Compute daily P&L per strategy: `SUM(CASE WHEN o.side='sell' THEN f.fill_price*f.fill_qty ELSE -f.fill_price*f.fill_qty END)` GROUP BY filled_at::date, o.strategy_id
   - Compute cumulative P&L via pandas cumsum
   - Also query `cmc_drift_metrics` for replay_pit_cumulative_pnl per config
   - Plotly go.Scatter traces: one for paper, one for replay per config
   - If no fills, return None (skip chart)

   **Chart 2: Drawdown chart** (`_build_drawdown_chart`)
   - From equity curve data, compute running peak and drawdown pct
   - drawdown = (cumulative_pnl - cumulative_pnl.cummax()) / abs(cumulative_pnl.cummax()) where cummax > 0
   - Plotly go.Scatter with fill='tozeroy' for drawdown area
   - If no fills, return None

   **Chart 3: Tracking error time series** (`_build_tracking_error_chart`)
   - Query `cmc_drift_metrics` for tracking_error_5d, tracking_error_30d by metric_date
   - Two traces (5d and 30d) + horizontal threshold line at 0.01 (1%)
   - Pattern from drift_report.py _plot_tracking_error()
   - If no drift metrics, return None

   **Chart 4: Slippage distribution** (`_build_slippage_chart`)
   - Query fill-level slippage bps: `ABS(f.fill_price::float - pb.open::float) / NULLIF(pb.open::float, 0) * 10000` from cmc_fills JOIN cmc_price_bars_multi_tf
   - Plotly go.Histogram with vertical line at 50 bps threshold
   - If no fills, return None

   **Chart 5: Kill switch event timeline** (`_build_kill_switch_timeline`)
   - Query `cmc_risk_events WHERE event_type LIKE 'kill_switch%' AND event_ts BETWEEN :start AND :end`
   - Plotly go.Scatter with markers showing events on a timeline
   - Different colors for activated vs disabled, exercise vs real
   - If no events, return None

   **Chart saving:** Use `_save_chart(fig, filename, charts_dir)` pattern from generate_bakeoff_scorecard.py (PNG via kaleido with HTML fallback).

{description + evidence + chart link if available}

## VAL-03: Slippage
{description + evidence + chart link if available}

{description + reference to kill switch exercise document}

## VAL-05: Log Audit
{description + audit summary + reference to audit report}

## Backtest Gates (from Phase 42)
{BT-01 and BT-02 with evidence sources}

{links to all generated charts, skip None charts}

{summary of tables queried and row counts}
