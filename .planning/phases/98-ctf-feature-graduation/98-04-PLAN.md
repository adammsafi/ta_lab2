Purpose: Identify cross-asset predictive relationships. If BTC's RSI slope predicts ETH's next-day return, that is actionable alpha for the BL optimizer and portfolio construction. This is the final research output of CTF feature graduation.

8. **Apply BH FDR correction** across ALL computed p-values:
   ```python

   pvals = [row['ic_p_value'] for row in all_results if row['ic_p_value'] is not None]
   reject, p_corrected, _, _ = multipletests(pvals, alpha=0.05, method='fdr_bh')
   # Map back: row['ic_p_bh'] = p_corrected[i], row['is_significant'] = reject[i]
   ```

9. **Persist to lead_lag_ic**: Use temp table + INSERT ON CONFLICT upsert:
   ```python
   df_results.to_sql("_tmp_lead_lag_ic", engine, schema="public", if_exists="replace", index=False)
   conn.execute(text("""
       INSERT INTO public.lead_lag_ic
           (asset_a_id, asset_b_id, feature, horizon, tf, venue_id, ic, ic_p_value, ic_p_bh, is_significant, n_obs)
       SELECT asset_a_id, asset_b_id, feature, horizon, tf, venue_id, ic, ic_p_value, ic_p_bh, is_significant, n_obs
       FROM public._tmp_lead_lag_ic
       ON CONFLICT (asset_a_id, asset_b_id, feature, horizon, tf, venue_id) DO UPDATE SET
           ic = EXCLUDED.ic, ic_p_value = EXCLUDED.ic_p_value, ic_p_bh = EXCLUDED.ic_p_bh,
   ```
