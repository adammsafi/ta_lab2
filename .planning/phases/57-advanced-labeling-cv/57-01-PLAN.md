    Create `src/ta_lab2/labeling/` package.

    **__init__.py:**
    Export key functions: `get_daily_vol`, `add_vertical_barrier`, `apply_triple_barriers`, `get_bins`.

    **triple_barrier.py** — implement from scratch (NO mlfinpy), following AFML Ch.3 patterns from RESEARCH.md:

    1. `get_daily_vol(close: pd.Series, span: int = 100) -> pd.Series`:
       - Compute log returns: `np.log(close / close.shift(1)).dropna()`
       - Return `log_ret.ewm(span=span).std()`
       - This is the vol measure used to scale barriers

    2. `add_vertical_barrier(t_events: pd.DatetimeIndex, close: pd.Series, num_bars: int = 5) -> pd.Series`:
       - For each event in t_events, find the index position in close, then go `num_bars` forward
       - Use `close.index.searchsorted(t_events)` to find positions, add `num_bars`, clamp to `len(close)-1`
       - Return `pd.Series(close.index[t1_positions], index=t_events[:len(t1_positions)])`
       - IMPORTANT: Use bar-count-based barriers, NOT calendar time (research pitfall #6)

    3. `apply_triple_barriers(close: pd.Series, t_events: pd.DatetimeIndex, pt_sl: list[float], target: pd.Series, num_bars: int = 5, side_prediction: pd.Series | None = None) -> pd.DataFrame`:
       - `pt_sl` = [pt_multiplier, sl_multiplier], e.g. [1.0, 1.0]
       - `target` = daily vol series (from get_daily_vol)
       - For each event t0 in t_events:
         a. Get entry_price = close.loc[t0]
         b. Get vol = target.loc[t0] (skip if NaN)
         c. Compute pt_barrier = entry_price * (1 + pt_sl[0] * vol)
         d. Compute sl_barrier = entry_price * (1 - pt_sl[1] * vol)
         e. Get vertical barrier t1 from add_vertical_barrier
         f. Search close[t0:t1] for first barrier touch
         g. If side_prediction is provided (meta-labeling mode): for side=+1 only upper barrier matters for profit; for side=-1 only lower barrier matters for profit
       - Return DataFrame with columns: ['t1', 'ret', 'bin', 'barrier_type']
         - t1: timestamp when barrier was hit
         - ret: return at barrier hit
         - bin: +1 (profit), -1 (stop), 0 (vertical/timeout)
         - barrier_type: 'pt', 'sl', 'vb'
       - Index = t_events (label start timestamps)

    4. `get_bins(triple_barrier_df: pd.DataFrame, close: pd.Series) -> pd.DataFrame`:
       - Convenience wrapper that takes apply_triple_barriers output
       - Computes final returns: `(close.loc[tb.t1].values / close.loc[tb.index].values) - 1`
       - Returns same DataFrame with `ret` column filled

    **Critical design decisions:**
    - All timestamps must be tz-aware UTC (for cv.py compatibility)
    - Use `pd.to_datetime(..., utc=True)` when constructing t1_series
    - The output `pd.Series(index=t0_timestamps, values=t1_timestamps)` IS the t1_series that PurgedKFoldSplitter/CPCVSplitter expects
    - Handle NaN in target (daily_vol) gracefully — skip events where vol is NaN
    - Handle edge case where vertical barrier extends past end of series — clamp to last available bar

    **Performance approach:** Start with a Python loop over events (correct first). Each event searches a slice of the close series for the first barrier touch using vectorized comparison on the slice. Do NOT attempt full O(n*L) numpy vectorization in this plan — correctness first.
