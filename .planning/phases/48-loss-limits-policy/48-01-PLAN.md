Create `src/ta_lab2/analysis/var_simulator.py` with pure library functions (no DB, no CLI). This module is the computation engine used by the CLI in Plan 02.

**Dataclass:**
```python
@dataclass
class VaRResult:
    strategy: str
    asset_id: int
    confidence: float
    historical_var: float
    parametric_var_normal: float
    cornish_fisher_var: float
    historical_cvar: float
    n_observations: int
    skewness: float
    excess_kurtosis: float
    cf_reliable: bool  # False if excess kurtosis > 8
```

**Functions (all operate on numpy arrays, no DB):**

1. `historical_var(returns: np.ndarray, confidence: float = 0.95) -> float`
   - `np.percentile(returns, (1 - confidence) * 100)`
   - Returns negative float (e.g., -0.05 = 5% loss)

2. `parametric_var_normal(returns: np.ndarray, confidence: float = 0.95) -> float`
   - `mu + z * sigma` where `z = norm.ppf(1 - confidence)`

3. `cornish_fisher_var(returns: np.ndarray, confidence: float = 0.95) -> float`
   - Cornish-Fisher expansion per Favre & Galeano (2002)
   - Use `scipy.stats.skew()` and `scipy.stats.kurtosis(fisher=True)` for excess kurtosis
   - z_cf = z + (z^2-1)*s/6 + (z^3-3z)*k/24 - (2z^3-5z)*s^2/36
   - Return `mu + z_cf * sigma`
   - Log WARNING if excess kurtosis > 8 (CF may be non-monotonic)

4. `historical_cvar(returns: np.ndarray, confidence: float = 0.95) -> float`
   - Mean of returns <= VaR (Expected Shortfall)
   - Return VaR itself if no returns below VaR (edge case)

5. `compute_var_suite(returns: np.ndarray, strategy: str, asset_id: int, confidence: float = 0.95) -> VaRResult`
   - Calls all 4 functions, populates VaRResult
   - Sets `cf_reliable = (abs(excess_kurtosis) <= 8)`

6. `var_to_daily_cap(var_results: list[VaRResult], method: str = "historical_95") -> float`
   - Translates VaR results to a daily loss cap
   - method = "historical_95" | "historical_99" | "cf_95" | "cf_99"
   - Returns `abs(median(var_values))` across results (median across strategies)
   - Sanity check: cap result at 0.15 (15%) max; log WARNING if computed value exceeds this

**Imports:** numpy, scipy.stats (norm, skew, kurtosis), dataclasses, logging, warnings

**Do NOT import:** sqlalchemy, pandas, vectorbt, plotly. This is a pure computation module.

import numpy as np
from ta_lab2.analysis.var_simulator import (
    historical_var, parametric_var_normal, cornish_fisher_var,

# Test with known normal distribution
rng = np.random.default_rng(42)
normal_returns = rng.normal(0, 0.02, 1000)  # mean=0, std=2%

h_var = historical_var(normal_returns, 0.95)
p_var = parametric_var_normal(normal_returns, 0.95)
cf_var = cornish_fisher_var(normal_returns, 0.95)
h_cvar = historical_cvar(normal_returns, 0.95)

print(f'Historical VaR 95%: {h_var:.4f}')
print(f'Parametric VaR 95%: {p_var:.4f}')
print(f'CF VaR 95%: {cf_var:.4f}')
print(f'CVaR 95%: {h_cvar:.4f}')

# For normal distribution, all 3 VaR methods should be close
assert abs(h_var - p_var) < 0.01, 'VaR methods diverge too much for normal data'
assert h_cvar < h_var, 'CVaR should be more negative than VaR'

# Test VaRResult
result = compute_var_suite(normal_returns, 'test_strategy', 1, 0.95)
assert isinstance(result, VaRResult)
assert result.cf_reliable is True  # normal dist has low kurtosis

# Test with fat-tailed distribution (should trigger CF warning)
fat_tail = np.concatenate([normal_returns, rng.standard_t(2, 100)])
result_fat = compute_var_suite(fat_tail, 'fat_tail', 1, 0.99)
print(f'Fat-tail excess kurtosis: {result_fat.excess_kurtosis:.1f}')
print(f'CF reliable: {result_fat.cf_reliable}')

# Test var_to_daily_cap
cap = var_to_daily_cap([result], method='historical_95')
