Create `src/ta_lab2/analysis/stop_simulator.py`. This module provides stop-loss simulation functions using vectorbt 0.28.1. It is a pure simulation library -- no DB reads, no CLI, no reports.

**Constants:**
```python
STOP_THRESHOLDS = [0.01, 0.03, 0.05, 0.07, 0.10, 0.15]
TIME_STOP_BARS = [5, 10, 20, 30]  # 5=~1 week, 10=~2 weeks, 20=~1 month, 30=~6 weeks
DEFAULT_FEE_BPS = 16  # 16 bps round-trip (consistent with bakeoff)
```

**Dataclass:**
```python
@dataclass
class StopScenarioResult:
    stop_type: str         # "hard", "trailing", "time"
    threshold: float       # percentage for hard/trailing, bar count for time
    sharpe: float
    max_dd: float
    total_return: float
    trade_count: int
    win_rate: float
    avg_recovery_bars: float  # NaN if no recovery events
    opportunity_cost: float   # total_return_baseline - total_return_with_stop
```

**Functions:**

1. `simulate_hard_stop(price, entries, exits, sl_pct, fee_bps=16) -> vbt.Portfolio`
   - `vbt.Portfolio.from_signals(price, entries=entries, exits=exits, sl_stop=sl_pct, sl_trail=False, direction="longonly", freq="D", init_cash=1000.0, fees=fee_bps/1e4)`
   - IMPORTANT: Strip timezone from price index before passing to vectorbt (tz boundary pattern from vbt_runner.py). Use `price.index = price.index.tz_localize(None)` if tz-aware.

2. `simulate_trailing_stop(price, entries, exits, sl_pct, fee_bps=16) -> vbt.Portfolio`
   - Same as hard stop but `sl_trail=True`

3. `simulate_time_stop(price, entries, exits, n_bars, fee_bps=16) -> vbt.Portfolio`
   - vectorbt 0.28.1 has NO native time-stop parameter. Build custom exit signal array.
   - For each True in entries, set `custom_exits.iloc[min(entry_idx + n_bars, len(price) - 1)] = True`
   - Combine with original exits via `custom_exits = exits | time_exits` (either original exit or time exit, whichever comes first)
   - Pass combined exits to `vbt.Portfolio.from_signals()` (no sl_stop/sl_trail params for time-stop)
   - IMPORTANT: Avoid Python for-loop over entries where possible. Use numpy vectorization: find entry indices, add n_bars, clip to array length, set exits.

4. `compute_recovery_time(equity: pd.Series) -> float`
   - Compute average bars from drawdown trough to equity recovery (return to peak)
   - Algorithm: Track cummax (peak equity). When equity < peak, we are in drawdown. When equity returns to peak, record bar count of the drawdown duration.
   - Return mean of all recovery durations. Return np.nan if equity never recovered (or no drawdowns).

5. `extract_scenario_metrics(pf: vbt.Portfolio, stop_type: str, threshold: float, baseline_return: float) -> StopScenarioResult`
   - Extract Sharpe via `pf.sharpe_ratio(freq=365)` (annualized for crypto 365-day year)
   - Extract max_dd via `float(pf.max_drawdown())`
   - Extract total_return via `float(pf.total_return())`
   - Extract trade_count via `int(pf.trades.count())`
   - Extract win_rate via `float(pf.trades.win_rate())` -- handle ZeroDivisionError if no trades (return 0.0)
   - Compute recovery time from `pf.value()` equity series via compute_recovery_time()
   - opportunity_cost = baseline_return - total_return

6. `sweep_stops(price, entries, exits, thresholds=None, time_bars=None, fee_bps=16) -> pd.DataFrame`
   - Default thresholds = STOP_THRESHOLDS, time_bars = TIME_STOP_BARS
   - First: run baseline (no stop) to get baseline_return for opportunity cost calculation
   - Then: for each threshold, simulate hard and trailing stops
   - Then: for each bar count in time_bars, simulate time-stop
   - For each: call extract_scenario_metrics() to build StopScenarioResult
   - Return DataFrame with columns: stop_type, threshold, sharpe, max_dd, total_return, trade_count, win_rate, avg_recovery_bars, opportunity_cost
   - Sort by (stop_type, threshold)

**Edge cases to handle:**
- Empty entries (no trades): return empty DataFrame with correct columns
- price with timezone: strip tz before vectorbt ingestion
- Entries/exits as boolean Series or boolean arrays: ensure both are boolean pd.Series with same index as price
- win_rate when trade_count == 0: return 0.0 (not NaN or error)

**Do NOT import:** sqlalchemy. This is a pure simulation module.

import pandas as pd
import numpy as np
from ta_lab2.analysis.stop_simulator import (
    simulate_hard_stop, simulate_trailing_stop, simulate_time_stop,
    compute_recovery_time, sweep_stops, extract_scenario_metrics,
    StopScenarioResult, STOP_THRESHOLDS, TIME_STOP_BARS,
)

# Create synthetic price data (trending up with volatility)
rng = np.random.default_rng(42)
n = 500
returns = rng.normal(0.001, 0.02, n)  # slight upward drift
price = pd.Series(100 * np.exp(np.cumsum(returns)), name='close')
price.index = pd.date_range('2020-01-01', periods=n, freq='D')

# Create simple entry/exit signals (every 30 bars)
entries = pd.Series(False, index=price.index)
exits = pd.Series(False, index=price.index)
entries.iloc[::30] = True
exits.iloc[15::30] = True  # exit 15 bars after entry

# Test individual stop types
pf_hard = simulate_hard_stop(price, entries, exits, 0.05)
print(f'Hard stop 5%: Sharpe={pf_hard.sharpe_ratio(freq=365):.2f}, trades={pf_hard.trades.count()}')

pf_trail = simulate_trailing_stop(price, entries, exits, 0.05)
print(f'Trailing stop 5%: Sharpe={pf_trail.sharpe_ratio(freq=365):.2f}, trades={pf_trail.trades.count()}')

pf_time = simulate_time_stop(price, entries, exits, 10)
print(f'Time stop 10 bars: Sharpe={pf_time.sharpe_ratio(freq=365):.2f}, trades={pf_time.trades.count()}')

# Test recovery time
equity = pf_hard.value()
rt = compute_recovery_time(equity)
print(f'Recovery time: {rt:.1f} bars')

# Test sweep
df = sweep_stops(price, entries, exits, thresholds=[0.03, 0.05, 0.10], time_bars=[5, 10])
print(f'Sweep shape: {df.shape}')
print(f'Sweep columns: {list(df.columns)}')
print(f'Stop types: {sorted(df.stop_type.unique())}')
assert 'hard' in df.stop_type.values
assert 'trailing' in df.stop_type.values
assert 'time' in df.stop_type.values
assert 'opportunity_cost' in df.columns

print('All stop simulator tests PASSED')
"
