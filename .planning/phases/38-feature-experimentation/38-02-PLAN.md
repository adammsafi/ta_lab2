  <action>
Create `src/ta_lab2/experiments/` directory with three files:

**registry.py** -- FeatureRegistry class:
- `__init__(self, yaml_path: str)`: Stores path, initializes empty _features dict.
- `load(self) -> None`: Opens YAML with `encoding='utf-8'`, calls `yaml.safe_load()`. Iterates `raw["features"]`, validates lifecycle (must be in {'experimental', 'promoted', 'deprecated'} -- raise ValueError if not), calls `_expand_params()` for each, stores in `self._features` dict keyed by expanded name. Raises `ValueError` on duplicate names after expansion.
- `_expand_params(self, name: str, spec: dict) -> list[dict]`: If no `params` key, returns single entry with `name` and `yaml_digest` added. If `params` exists, uses `itertools.product` to expand all param combinations. For each combo: builds `variant_name = f"{name}_{'_'.join(f'{k}{v}' for k, v in variant_params.items())}"`. If mode is inline, substitutes params into expression via `.format(**variant_params)`. Adds `yaml_digest` and `resolved_params` to each variant.
- `_digest(self, spec: dict) -> str`: `hashlib.sha256(json.dumps(spec, sort_keys=True, default=str).encode('utf-8')).hexdigest()[:16]`
- `get_feature(self, name: str) -> dict`: Returns self._features[name] (KeyError if missing).
- `list_experimental(self) -> list[str]`: Returns names where lifecycle == 'experimental'.
- `list_all(self) -> dict[str, dict]`: Returns full _features dict.
- `validate_expression(self, expr: str) -> None`: Calls `ast.parse(expr, mode='eval')`, catches SyntaxError and re-raises as ValueError with descriptive message.
- `validate_dotpath(self, dotpath: str) -> None`: Checks ':' separator exists, splits into module_path and func_name, calls `importlib.import_module(module_path)` and `hasattr(mod, func_name)`. Raises ValueError on ImportError or missing attr.
- Call `validate_expression` or `validate_dotpath` during `load()` for each feature based on compute mode.

**dag.py** -- DAG resolution:
- `resolve_experiment_dag(features: dict[str, dict]) -> list[str]`: Builds dependency dict from `spec.get("depends_on", [])`. Uses `graphlib.TopologicalSorter(deps)` and returns `list(sorter.static_order())`. Raises `graphlib.CycleError` on circular deps. Filter to only include features that exist in the input dict (skip references to promoted/external features not in registry).

**__init__.py** -- Package exports:
- Import and export: `FeatureRegistry` from `.registry`, `resolve_experiment_dag` from `.dag`.

IMPORTANT:
- Use `encoding='utf-8'` when opening YAML (Windows pitfall).
- Do NOT install new packages.
- Inject `{"np": numpy, "pd": pandas, "__builtins__": {}}` as eval globals (documented in research open question 3).

```yaml
# Feature Experimentation Registry
# Features with lifecycle: experimental are scored by ExperimentRunner
# Promoted features are tracked in dim_feature_registry (DB)
#
# Compute modes:
#   inline: pandas expression evaluated with DataFrame columns as locals
#   dotpath: Python function reference as module.path:function_name

features:
  vol_ratio_30_7:
    lifecycle: experimental
    description: "30d/7d realized vol ratio -- captures vol regime shift"
    compute:
      mode: inline
      expression: "vol_30d / (vol_7d + 1e-10)"
    inputs:
      - table: cmc_vol
        columns: [vol_30d, vol_7d]
    tags: [vol, ratio]

  rsi_momentum:
    lifecycle: experimental
    description: "RSI-based momentum signal"
    compute:
      mode: inline
      expression: "rsi_14 - 50"
    inputs:
      - table: cmc_ta_daily
        columns: [rsi_14]
    tags: [momentum, rsi]

  ret_vol_ratio:
    lifecycle: experimental
    description: "Return-to-volatility ratio across param sweep"
    compute:
      mode: inline
      expression: "close.pct_change({period}) / (close.pct_change({period}).rolling({period}).std() + 1e-10)"
    params:
      period: [5, 14, 21]
    inputs:
      - table: cmc_price_bars_multi_tf_u
        columns: [close]
    tags: [sharpe, momentum]
```

The YAML must be loadable by FeatureRegistry.load() without errors. The ret_vol_ratio entry demonstrates parameter sweep expansion into 3 variants.
