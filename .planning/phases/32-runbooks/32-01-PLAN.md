2. **Prerequisites** — What must be true:
   - DB up and `TARGET_DB_URL` set

   - Individual signal type scripts (refresh_cmc_signals_rsi_mean_revert, etc.)

   - Realistic mode (--fee-bps, --slippage-bps, --save-results)
   - JSON output mode (--output-json)

6. **Result Tables** — Describe 3 backtest tables (cmc_backtest_runs, cmc_backtest_trades, cmc_backtest_metrics) with key columns from research.
