  <action>
Add two functions to the existing param_optimizer.py (created in 105-01):

1. **`plateau_score(trial_results, best_params, threshold=0.80, neighbor_radius=2)`**
   - `trial_results`: list of dicts with keys `"params"` (dict) and `"ic"` (float or None)
   - `best_params`: dict of parameter_name -> value for the best trial
   - Finds `best_ic = max(r["ic"] for valid r)`. If best_ic <= 0, return 0.0.
   - `min_ic = threshold * best_ic`
   - For single-parameter indicators (e.g., RSI window): a neighbor is any trial where `abs(p[k] - best_params[k]) <= neighbor_radius` for ALL parameter keys
   - For multi-parameter indicators (e.g., MACD fast/slow/signal): normalize each param dimension to [0,1] using `(value - min_val) / (max_val - min_val)` across all trial values. A neighbor is any trial where L-infinity distance in normalized space <= `neighbor_radius / max_range` where `max_range = max(high - low)` across all params. This keeps scale-invariant.
   - Return `float(count(neighbor IC >= min_ic) / count(neighbors))`
   - Returns float in [0.0, 1.0]

2. **`rolling_stability_test(feature_series, fwd_ret, train_start, train_end, n_windows=5, max_sign_flips=1, max_ic_cv=2.0, min_obs_per_window=50)`**
   - `feature_series`: pd.Series (precomputed indicator values for best_params)
   - `fwd_ret`: pd.Series (forward returns)
   - Split [train_start, train_end] range into `n_windows` equal non-overlapping chunks using `np.array_split` on the sorted index
   - For each chunk: compute Spearman IC between feature and fwd_ret (using `scipy.stats.spearmanr` directly, NOT compute_rolling_ic which uses overlapping windows)
   - Skip windows with < min_obs_per_window valid observations after dropna
   - Count sign_flips: number of windows where IC sign differs from median IC sign across all windows
   - Compute ic_cv = std(ics, ddof=1) / abs(mean(ics)). If abs(mean) < 1e-10, ic_cv = inf
   - passes = (sign_flips <= max_sign_flips) AND (ic_cv <= max_ic_cv) AND (n_valid_windows >= 3)
   - Return dict: `{"passes": bool, "sign_flips": int, "ic_cv": float, "window_ics": list[float], "n_valid_windows": int}`

Both functions are pure computation -- no DB access. Type hints throughout.
  </action>

from ta_lab2.analysis.param_optimizer import plateau_score, rolling_stability_test
import pandas as pd, numpy as np

# plateau_score: broad plateau
results = [{'params': {'window': w}, 'ic': 0.05 - 0.001 * abs(w - 14)} for w in range(5, 31)]
ps = plateau_score(results, {'window': 14})
assert 0.5 < ps <= 1.0, f'Expected broad plateau, got {ps}'

# plateau_score: sharp peak
results2 = [{'params': {'window': w}, 'ic': 0.05 if w == 14 else 0.01} for w in range(5, 31)]
ps2 = plateau_score(results2, {'window': 14})
assert ps2 < ps, f'Sharp peak {ps2} should be < broad plateau {ps}'

# rolling_stability: stable signal
idx = pd.date_range('2020-01-01', periods=1000, freq='D', tz='UTC')
np.random.seed(42)
feat = pd.Series(np.cumsum(np.random.randn(1000)), index=idx)
fwd = pd.Series(feat.values * 0.01 + np.random.randn(1000)*0.005, index=idx)
res = rolling_stability_test(feat, fwd, idx[0], idx[-1])
assert res['n_valid_windows'] >= 3
print(f'plateau_score broad={ps:.3f} sharp={ps2:.3f}')
print(f'stability: passes={res[\"passes\"]}, flips={res[\"sign_flips\"]}, cv={res[\"ic_cv\"]:.2f}')
"

   - `feature_best`: pd.Series (indicator values for the best parameter set)
   - `fwd_ret`: pd.Series (forward returns)
   - `all_sweep_ics`: list[float] (IC value for EVERY parameter combination tested in the sweep)
   - Compute rolling IC series using `compute_rolling_ic(feature_best, fwd_ret, window=window)` from ic.py
   - Clean: `rolling_ic_clean = rolling_ic.dropna().values`
   - If len(rolling_ic_clean) < 30: return `{"dsr": float("nan"), "n_trials": len(all_sweep_ics), "note": "insufficient_rolling_ic"}`
   - Filter valid ICs: `valid_ics = [ic for ic in all_sweep_ics if ic is not None and not np.isnan(ic)]`
   - Call `compute_dsr(best_trial_returns=rolling_ic_clean, sr_estimates=valid_ics)` from psr.py
   - Return dict: `{"dsr": float, "n_trials": len(valid_ics), "rolling_ic_n": len(rolling_ic_clean)}`
   - Import: `from ta_lab2.backtests.psr import compute_dsr` and `from ta_lab2.analysis.ic import compute_rolling_ic`

2. **`select_best_from_sweep(sweep_result, feature_fn, close, high, low, volume, fwd_ret, train_start, train_end, tf_days_nominal, conn=None, top_n=5)`**
   - `sweep_result`: dict returned by run_sweep() (has trials, sweep_id, etc.)
   - Orchestrates the full selection pipeline:
     a. Build trial_results list from sweep_result["trials"] (COMPLETE trials only): `[{"params": t.params, "ic": t.value} for t in trials if t.state == COMPLETE and t.value is not None]`
     b. Sort by IC descending, take top_n candidates
     c. For each candidate: compute plateau_score()
     d. Select the candidate with the highest plateau_score (tie-break: highest IC)
     e. Compute feature_best using feature_fn(**selected_params) on the full data
     f. Run rolling_stability_test(feature_best, fwd_ret, train_start, train_end)
     g. Compute compute_dsr_over_sweep(feature_best, fwd_ret, all_sweep_ics)
     h. If conn provided: UPDATE trial_registry SET plateau_score=X, rolling_stability_passes=Y, ic_cv=Z, sign_flips=W, dsr_adjusted_sharpe=D, n_sweep_trials=N WHERE sweep_id=S AND param_set=str(selected_params)
   - Return dict: `{"selected_params": dict, "ic": float, "plateau_score": float, "stability": dict (from rolling_stability_test), "dsr": dict (from compute_dsr_over_sweep), "sweep_id": str}`

This function ties together all the overfitting-aware selection logic into a single callable pipeline.
  </action>

from ta_lab2.analysis.param_optimizer import (
    run_sweep, select_best_from_sweep, compute_dsr_over_sweep

import pandas as pd, numpy as np

# End-to-end test with dummy data
idx = pd.date_range('2024-01-01', periods=500, freq='D', tz='UTC')
np.random.seed(42)

    fwd_ret=fwd, train_start=idx[0], train_end=idx[-2],

result = select_best_from_sweep(

    idx[0], idx[-2], tf_days_nominal=1,
)
assert result['selected_params'] is not None
assert 0.0 <= result['plateau_score'] <= 1.0
assert 'passes' in result['stability']
print(f'Selected: {result[\"selected_params\"]}, plateau={result[\"plateau_score\"]:.3f}')
print(f'Stability: passes={result[\"stability\"][\"passes\"]}, DSR={result[\"dsr\"][\"dsr\"]:.4f}')
"
