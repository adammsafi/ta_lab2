  <files>src/ta_lab2/analysis/param_optimizer.py</files>
  <action>
Create `src/ta_lab2/analysis/param_optimizer.py` with these functions:

1. **`_make_ic_objective(feature_fn, close, high, low, volume, fwd_ret, train_start, train_end, param_space_def, tf_days_nominal, min_obs=50)`**
   - Returns an Optuna objective callable
   - Inside the objective: extract params via `_suggest_params(trial, param_space_def)`, call `feature_fn(close=close, high=high, low=low, volume=volume, **params)` to compute indicator, then compute Spearman IC vs fwd_ret in [train_start, train_end] window
   - Boundary-mask: set fwd_ret to NaN where `feat.index + timedelta(days=tf_days_nominal) > train_end`
   - For constraint violations (e.g., MACD fast >= slow): `raise optuna.exceptions.TrialPruned()`
   - If insufficient valid observations (< min_obs after dropna): return `float("nan")`
   - Return IC as float (study direction="maximize")

2. **`_suggest_params(trial, param_space_def)`**
   - `param_space_def` is a list of dicts: `[{"name": "window", "type": "int", "low": 5, "high": 30}, {"name": "std", "type": "float", "low": 1.0, "high": 3.0, "step": 0.25}]`
   - Maps to `trial.suggest_int(name, low, high)` or `trial.suggest_float(name, low, high, step=step)`
   - Returns dict of param_name -> value

3. **`_log_sweep_to_registry(conn, trials, sweep_id, indicator_name, asset_id, tf, venue_id=1)`**
   - Takes list of Optuna FrozenTrial objects + metadata
   - For each COMPLETE trial: build row dict mapping to trial_registry columns:
     - indicator_name, param_set=str(trial.params), tf, asset_id, venue_id
     - ic_observed=trial.value, sweep_id=sweep_id, horizon=1, return_type='arith'
     - source_table='param_sweep'
   - Upsert via temp table + ON CONFLICT DO UPDATE on (indicator_name, param_set, tf, asset_id, venue_id, horizon, return_type) -- update ic_observed, sweep_id, n_sweep_trials, sweep_ts=now()
   - Return count of rows written

4. **`run_sweep(indicator_name, feature_fn, param_space_def, close, high, low, volume, fwd_ret, train_start, train_end, asset_id, tf, tf_days_nominal, venue_id=1, grid_threshold=200, tpe_n_trials=100, seed=42, conn=None)`**
   - Compute grid_size from param_space_def (product of all range sizes)
   - If grid_size <= grid_threshold: use `optuna.samplers.GridSampler` with explicit Python int/float value lists built from param_space_def. CRITICAL: use `list(range(low, high+1))` for int params, NOT np.arange. For float params with step: `[round(low + i*step, 4) for i in range(int((high-low)/step)+1)]`
   - If grid_size > grid_threshold: use `optuna.samplers.TPESampler(seed=seed, multivariate=True)` with n_trials=tpe_n_trials
   - `optuna.logging.set_verbosity(optuna.logging.WARNING)` to suppress per-trial output
   - Create in-memory study (direction="maximize"), run optimize
   - Generate sweep_id = str(uuid.uuid4())
   - If conn provided: call _log_sweep_to_registry
   - Return dict: `{"sweep_id": sweep_id, "best_params": study.best_params, "best_ic": study.best_value, "n_trials": len(study.trials), "n_complete": len([t for t in study.trials if t.state == optuna.trial.TrialState.COMPLETE]), "trials": study.trials}`

Suppress Optuna output with `optuna.logging.set_verbosity(optuna.logging.WARNING)` at module level.

Add module docstring explaining this is an IC-based parameter optimizer (NOT Sharpe-based like parameter_sweep.py, NOT ML-based like run_optuna_sweep.py).

Type hints throughout. Use `from __future__ import annotations`.
