<task type="auto">
  <name>Task 2: CLI main loop and argparse</name>
  <files>src/ta_lab2/scripts/analysis/run_param_sweep.py</files>
  <action>
Add the CLI entry point to run_param_sweep.py:

1. **argparse** with flags:
   - `--indicator`: sweep a single indicator by name (skip survivor lookup)
   - `--asset-id`: sweep a single asset (default: all assets with IC data)
   - `--tf`: timeframe filter (default: '1D')
   - `--venue-id`: venue filter (default: 1)
   - `--dry-run`: list indicators and parameter spaces without running sweeps
   - `--top-n`: number of top-IC candidates to evaluate for plateau (default: 5)
   - `--tpe-trials`: number of TPE trials for large spaces (default: 100)
   - `--skip-stability`: skip rolling_stability_test (useful for quick exploration)
   - `--skip-dsr`: skip DSR computation

2. **main()** flow:
   a. Parse args, set up logging (INFO level)
   b. Create DB connection (from db_config.env or TARGET_DB_URL, using project get_engine() pattern)
   c. Determine indicator list:
      - If --indicator: use just that one (must be in PARAM_SPACE_REGISTRY)
      - Else: call _load_survivors(conn) and intersect with PARAM_SPACE_REGISTRY keys
   d. Determine asset list:
      - If --asset-id: use just that one
      - Else: query `SELECT DISTINCT id FROM ic_results WHERE regime_col='all' LIMIT 20` (cap at 20 for initial runs)
   e. If --dry-run: print table of (indicator, grid_size, sampler_type, param_space) and exit
   f. For each (indicator, asset_id, tf):
      - Resolve feature_fn via _resolve_feature_fn. Skip if None.
      - Load data via _load_data_for_asset. Skip if insufficient data.
      - Determine train_start (min ts) and train_end (max ts minus 1 period)
      - Call run_sweep(..., conn=conn)
      - Call select_best_from_sweep(..., conn=conn) unless --skip-stability and --skip-dsr
      - Log result: indicator, asset_id, tf, best_params, IC, plateau_score, stability passes, DSR
   g. Print summary table at end: total sweeps, indicators swept, trials run, time elapsed

3. **`if __name__ == "__main__": main()`**

Use NullPool for the engine (project convention for scripts). Add encoding='utf-8' for any file I/O.

Handle keyboard interrupt gracefully (log partial results, don't crash).

Log format: `%(asctime)s %(levelname)s %(message)s`
