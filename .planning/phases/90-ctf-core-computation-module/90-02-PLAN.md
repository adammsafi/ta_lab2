**1. Module-level helper: _compute_slope(series, window)**
Following the expression_engine.py pattern (raw=True for speed):
```python
def _compute_slope(series: pd.Series, window: int) -> pd.Series:
    n = int(window)
    x = np.arange(n, dtype=float)
    x_mean = x.mean()
    x_denom = ((x - x_mean) ** 2).sum()

    def _apply(arr: np.ndarray) -> float:
        y_mean = arr.mean()
        return float(np.dot(arr - y_mean, x - x_mean) / x_denom)

    return series.rolling(window=n, min_periods=2).apply(_apply, raw=True)
```
Slope is computed on the base_value series (the finer timeframe), NOT the ref_value.

**2. Module-level helper: _compute_divergence(base_series, ref_series, window)**
```python
def _compute_divergence(base_series, ref_series, window) -> pd.Series:
    diff = base_series - ref_series
    std = base_series.rolling(window=window, min_periods=window // 2).std()
    return diff / std.where(std > 1e-12, other=np.nan)
```

**3. Module-level helper: _compute_agreement(base_series, ref_series, is_directional, window=20)**
- For `is_directional=True`: `agree = (np.sign(base_series) * np.sign(ref_series)) > 0`
  (signs of the values themselves agree)
- For `is_directional=False`: `agree = (np.sign(base_series.diff()) * np.sign(ref_series.diff())) > 0`
  (both moving in the same direction)
- Return: `agree.astype(float).rolling(window=window, min_periods=max(5, window // 3)).mean()`
- Window defaults to 20 (hardcoded default, may be exposed to YAML in Phase 91).

**4. Module-level helper: _compute_crossover(base_series, ref_series, is_directional)**
- For `is_directional=False`: return `pd.Series(np.nan, index=base_series.index)`
  (crossover is meaningless for oscillators)
- For `is_directional=True`:
  ```python
  prev_above = base_series.shift(1) > ref_series.shift(1)
  curr_above = base_series > ref_series
  crossed_up = (~prev_above) & curr_above
  crossed_dn = prev_above & (~curr_above)
  return crossed_up.astype(float) - crossed_dn.astype(float)
  ```
  Result: +1 = crossed above, -1 = crossed below, 0 = no crossover.

g) For each indicator in source_indicators:

     - df_a["agreement"] = _compute_agreement(df_a["base_value"], df_a["ref_value"], is_dir)
     - df_a["crossover"] = _compute_crossover(df_a["base_value"], df_a["ref_value"], is_dir)
     - Append to results list
   - Concat per-indicator results, add columns: venue_id, base_tf, ref_tf, indicator_id, alignment_source
   - Rename base_value/ref_value to match ctf table columns (base_value, ref_value are the ctf column names)
   - Write via _write_to_db, accumulate row count

     base_tf = tf_pair["base_tf"]
