**6. _load_indicators_batch() method:**
Parameters: `ids: list[int], source_table: str, columns: list[str], tf: str, extra_filter: str = ""`

CRITICAL source table asymmetries (from research):
- `returns_bars_multi_tf_u`: ts column is `"timestamp"` (quoted reserved word, NOT `ts`).
  Alias it as `ts` in the SELECT. Also requires `AND roll = FALSE` extra_filter.
- `ta` and `vol`: ts column is `ts`. venue_id is column-only (NOT in PK). No extra filter.
- `features`: ts column is `ts`. venue_id IS in PK. Add `AND venue_id = :venue_id` to WHERE.

Build SQL dynamically:
- `ts_col = '"timestamp"' if source_table == "returns_bars_multi_tf_u" else "ts"`
- `col_list = ", ".join(f'"{c}"' for c in columns)` (quote all column names for safety)
- For `features` table: append `AND venue_id = :venue_id` to WHERE clause
- Use `text()` for the query with `:ids`, `:tf`, `:as_` params (and `:venue_id` for features)
- After pd.read_sql: `df["ts"] = pd.to_datetime(df["ts"], utc=True)` (CRITICAL Windows tz fix)
- ORDER BY id, ts ASC in the SQL

**7. _align_timeframes() method:**
Parameters: `base_df: pd.DataFrame, ref_df: pd.DataFrame, source_col: str`

For each asset_id in base_df["id"].unique():
- Extract base_asset = base_df[base_df["id"] == asset_id]
- Extract ref_asset = ref_df[ref_df["id"] == asset_id]
- If ref_asset is empty, skip (no ref data for this asset)
- Call `build_alignment_frame(low_df=base_asset[["ts", source_col]], high_df=ref_asset[["ts", source_col]], on="ts", low_cols=[source_col], high_cols=[source_col], suffix_low="", suffix_high="_ref", direction="backward")`
- Rename columns: base_value = source_col, ref_value = f"{source_col}_ref"
- Add "id" column back to the aligned frame
- Collect per-asset aligned frames, pd.concat at end

Return DataFrame with columns: [id, ts, base_value, ref_value]
