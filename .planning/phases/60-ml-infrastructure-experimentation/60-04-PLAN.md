  Create `src/ta_lab2/ml/feature_importance.py` with these functions:

  1. `compute_mda(model, X: pd.DataFrame, y: np.ndarray, t1_series: pd.Series, n_splits: int = 5, n_repeats: int = 10, scoring: str = 'accuracy') -> pd.Series`:
     - Create PurgedKFoldSplitter with n_splits, t1_series, embargo_frac=0.01
     - For each (train_idx, test_idx) from cv.split(X.values):
       - CRITICAL: `if len(train_idx) == 0 or len(test_idx) == 0: continue` (empty fold guard)
       - Clone model, fit on X.iloc[train_idx], y[train_idx]
       - Call `sklearn.inspection.permutation_importance(model, X.iloc[test_idx], y[test_idx], n_repeats=n_repeats, random_state=42, scoring=scoring)`
       - Collect `result.importances_mean` per fold
     - Average across folds, return as pd.Series sorted descending
     - If no valid folds, return pd.Series of zeros with X.columns as index
     - CRITICAL: Always pass DataFrame (not numpy array) to model.fit and permutation_importance to avoid LightGBM feature name warnings

  2. `compute_sfi(model, X: pd.DataFrame, y: np.ndarray, t1_series: pd.Series, n_splits: int = 5, scoring: str = 'accuracy') -> pd.Series`:
     - For each column in X:
       - Create single-column DataFrame: `X_single = X[[col]]`
       - Create PurgedKFoldSplitter
       - For each fold: empty fold guard, clone model, fit, score with `accuracy_score`
       - Average fold scores for this column
     - Return as pd.Series sorted descending
     - CRITICAL: Always pass DataFrame slices (not numpy arrays) to model

  3. `cluster_features(X: pd.DataFrame, threshold: float = 0.5) -> dict[str, list[str]]`:
     - Compute Spearman correlation matrix: `scipy.stats.spearmanr(X).statistic`
     - Symmetrize: `corr = (corr + corr.T) / 2; np.fill_diagonal(corr, 1)`
     - Distance matrix: `1 - np.abs(corr)`
     - Ward linkage: `scipy.cluster.hierarchy.ward(squareform(distance_matrix))`
     - Cluster assignment: `scipy.cluster.hierarchy.fcluster(linkage, t=threshold, criterion='distance')`
     - Return dict mapping `cluster_{id}` to list of feature names
     - Handle edge case: if X has 1 column, return single cluster

  4. `compute_clustered_mda(model, X: pd.DataFrame, y: np.ndarray, t1_series: pd.Series, n_splits: int = 5, n_repeats: int = 10, scoring: str = 'accuracy', cluster_threshold: float = 0.5) -> pd.DataFrame`:
     - First cluster features using cluster_features()
     - For each cluster, compute MDA by permuting ALL features in the cluster simultaneously
     - Return DataFrame with columns: [cluster_id, features, importance_mean]
     - Implementation: for each fold, for each cluster, shuffle all columns in that cluster at once, measure accuracy drop
