  Run an end-to-end test of the expression engine through FeatureRegistry:

  1. Load `configs/experiments/features.yaml` through FeatureRegistry
  2. Verify expression-mode features are loaded (count >= 5)
  3. Create a small synthetic DataFrame with OHLCV columns
  4. For each expression-mode feature:
     - Extract the expression from the loaded spec
     - Call `evaluate_expression(expression, df)` directly
     - Verify it returns a pd.Series of the correct length
  5. Print summary: N expression features validated, all producing correct output shapes

  This is a verification-only task -- no files are created or modified. It confirms Plans 01+03 work together.

  ```python
  import pandas as pd
  import numpy as np
  from ta_lab2.experiments.registry import FeatureRegistry
  from ta_lab2.ml.expression_engine import evaluate_expression

  reg = FeatureRegistry('configs/experiments/features.yaml')
  reg.load()

  expr_features = {k: v for k, v in reg.features.items()
                   if v.get('compute', {}).get('mode') == 'expression'}
  print(f'Expression features: {len(expr_features)}')
  assert len(expr_features) >= 5

  df = pd.DataFrame({
      'close': np.random.lognormal(4, 0.5, 100),
      'open': np.random.lognormal(4, 0.5, 100),
      'high': np.random.lognormal(4, 0.5, 100),
      'low': np.random.lognormal(4, 0.5, 100),
      'volume': np.random.randint(1000, 10000, 100).astype(float),
  })

  for name, spec in expr_features.items():
      expr = spec['compute']['expression']
      try:
          result = evaluate_expression(expr, df)
          assert isinstance(result, pd.Series)
          assert len(result) == len(df)
          print(f'  OK: {name} -> {result.dropna().shape[0]} non-null values')
      except Exception as e:
          print(f'  FAIL: {name} -> {e}')
          raise

  print('All expression features validated end-to-end')
  ```
