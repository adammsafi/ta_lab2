  Create `src/ta_lab2/ml/double_ensemble.py` with class `DoubleEnsemble`:

  Module docstring: explain the DoubleEnsemble concept -- sliding-window ensemble where each sub-model trains on a different time window, with sample reweighting to upweight difficult samples, and recency-weighted prediction aggregation. Reference Qlib DoubleEnsemble paper.

  Constructor `__init__(self, window_size: int = 60, stride: int = 15, base_params: dict | None = None)`:
  - `self.window_size = window_size`
  - `self.stride = stride`
  - `self.base_params` defaults to `{'n_estimators': 100, 'num_leaves': 20, 'learning_rate': 0.05, 'verbose': -1}`
  - `self.models: list[tuple[Any, float]] = []` -- (model, recency_weight) pairs (LGBMClassifier resolved at runtime via lazy import)
  - `self.classes_: np.ndarray | None = None`

  Methods:

  1. `fit(self, X: pd.DataFrame, y: np.ndarray) -> 'DoubleEnsemble'`:
     - Validate: X must be a DataFrame (not numpy array) -- LightGBM feature name pitfall
     - `n = len(X)`
     - `self.models = []`
     - `self.classes_ = np.unique(y)`
     - For `start` in `range(0, n - self.window_size + 1, self.stride)`:
       - `end = start + self.window_size`
       - `X_win, y_win = X.iloc[start:end], y[start:end]`
       - If len(unique(y_win)) < 2: skip window (single class -- can't train classifier)
       - Round 1: train base LGBMClassifier on X_win, y_win
       - Compute sample weights: `_compute_sample_weights(clf1, X_win, y_win)`
       - Round 2: train new LGBMClassifier on X_win, y_win with sample_weight=sw
       - Compute recency weight: `end / n` (later windows weighted higher)
       - Append `(clf2, recency_weight)` to self.models
     - If no models trained (data too short), train single global model on all data
     - Log number of sub-models trained at INFO level
     - Return self

  2. `_compute_sample_weights(self, clf, X: pd.DataFrame, y: np.ndarray) -> np.ndarray`:
     - Get predict_proba from clf on X
     - Confidence = |proba[:, 1] - 0.5| (how confident the model is)
     - Weights = 1.0 - confidence (upweight uncertain samples)
     - Normalize: `weights = weights / weights.sum() * len(X)`
     - Return weights

  3. `predict_proba(self, X: pd.DataFrame) -> np.ndarray`:
     - Validate: X must be DataFrame
     - Collect recency weights from self.models
     - Normalize weights: `weights / weights.sum()`
     - Weighted average of predict_proba from each sub-model
     - Return (n_samples, n_classes) array

  4. `predict(self, X: pd.DataFrame) -> np.ndarray`:
     - Call predict_proba, return argmax mapped to self.classes_

  5. `get_model_info(self) -> dict`:
     - Return dict with n_models, window_size, stride, base_params

  Handle edge cases:
  - If X has fewer rows than window_size: train single global model
  - If y has single class in a window: skip that window
  - If no models produced: raise ValueError with helpful message

  Import: Use LAZY import for lightgbm -- do NOT import at module top level. Instead, import inside `fit()` and `predict_proba()` methods: `import lightgbm as lgb`. This ensures the module is importable even before lightgbm is installed (Plan 60-08 installs it in Wave 3). If lightgbm is not installed when fit() is called, raise ImportError with message "pip install lightgbm==4.6.0".
