  <files>src/ta_lab2/ml/__init__.py, src/ta_lab2/ml/expression_engine.py</files>
  <action>
  Create `src/ta_lab2/ml/__init__.py` (empty, just docstring: "ML infrastructure for ta_lab2.").

  Create `src/ta_lab2/ml/expression_engine.py` with:

  1. `OPERATOR_REGISTRY` dict mapping operator names to lambda/functions:
     - `EMA(series, n)` -> `series.ewm(span=int(n), adjust=False).mean()`
     - `Ref(series, n)` -> `series.shift(int(n))`
     - `Delta(series, n)` -> `series - series.shift(int(n))`
     - `Mean(series, n)` -> `series.rolling(int(n), min_periods=1).mean()`
     - `Std(series, n)` -> `series.rolling(int(n), min_periods=1).std()`
     - `WMA(series, n)` -> weighted moving average using `np.dot` on rolling window
     - `Max(series, n)` -> `series.rolling(int(n), min_periods=1).max()`
     - `Min(series, n)` -> `series.rolling(int(n), min_periods=1).min()`
     - `Rank(series)` -> `series.rank(pct=True)` (no window param)
     - `Abs(series)` -> `series.abs()` (no window param)
     - `Sign(series)` -> `np.sign(series)` (no window param)
     - `Log(series)` -> `np.log(series.clip(lower=1e-10))` (no window param)
     - `Corr(series1, series2, n)` -> `series1.rolling(int(n)).corr(series2)`
     - `Slope(series, n)` -> linear regression slope over rolling window of n bars
     - `Skew(series, n)` -> `series.rolling(int(n)).skew()`
     - `Kurt(series, n)` -> `series.rolling(int(n)).kurt()`

  2. `validate_expression(expression: str, allowed_columns: list[str] | None = None) -> None`:
     - Replace `$col` references with `_placeholder_` via regex
     - Validate with `ast.parse(parsed, mode='eval')`
     - If `allowed_columns` provided, check all `$col` references are in the allowlist
     - Raise `ValueError` on invalid syntax or disallowed columns

  3. `evaluate_expression(expression: str, df: pd.DataFrame) -> pd.Series`:
     - Replace `$col` with `_df_['col']` via `re.sub(r'\$(\w+)', ...)`
     - Build safe globals: `{'__builtins__': {}, 'np': np, 'pd': pd}` plus all OPERATOR_REGISTRY entries
     - Build local vars: `{'_df_': df}` plus all DataFrame columns by name
     - Call `eval(parsed, safe_globals, local_vars)`
     - Return result as pd.Series (coerce if needed)
     - Raise `ValueError` on evaluation failure with clear error message

  Security: restrict `__builtins__` to empty dict. Do NOT allow arbitrary Python builtins.
  Follow the exact pattern from 60-RESEARCH.md Pattern 1.
  </action>
