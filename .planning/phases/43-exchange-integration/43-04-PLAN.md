**canonical_order.py:**

Create a `CanonicalOrder` dataclass (not frozen -- needs mutation for signal_id assignment):

Fields:
- `pair: str` -- normalized as "BTC/USD" (slash-separated)
- `side: Literal["buy", "sell"]`
- `order_type: Literal["market", "limit", "stop"]`
- `quantity: float` -- base asset quantity
- `limit_price: Optional[float] = None`
- `stop_price: Optional[float] = None`
- `client_order_id: str = field(default_factory=lambda: str(uuid.uuid4()))`
- `signal_id: Optional[int] = None` -- FK to cmc_signals if originated from signal pipeline
- `asset_id: Optional[int] = None` -- FK to dim_assets

Methods:

`validate(self)` -- raises ValueError if:
- order_type is "limit" and limit_price is None
- order_type is "stop" and stop_price is None
- quantity <= 0
- side not in ("buy", "sell")
- order_type not in ("market", "limit", "stop")

`to_exchange(self, exchange: str) -> dict`:
- Dispatch to `_to_coinbase()` or `_to_kraken()` based on exchange name (case-insensitive)
- Raise ValueError for unknown exchange

`_to_coinbase(self) -> dict`:
- product_id: pair.replace("/", "-").upper() (e.g., "BTC-USD")
- For market order:
  ```python
  {
      "client_order_id": self.client_order_id,
      "product_id": product_id,
      "side": self.side.upper(),
      "order_configuration": {
          "market_market_ioc": {
              "base_size": str(self.quantity)  # Always use base_size for paper
          }
      }
  }
  ```
- For limit order:
  ```python
  {
      "client_order_id": self.client_order_id,
      "product_id": product_id,
      "side": self.side.upper(),
      "order_configuration": {
          "limit_limit_gtc": {
              "base_size": str(self.quantity),
              "limit_price": str(self.limit_price)
          }
      }
  }
  ```
- For stop order:
  ```python
  {
      "client_order_id": self.client_order_id,
      "product_id": product_id,
      "side": self.side.upper(),
      "order_configuration": {
          "stop_limit_stop_limit_gtc": {
              "base_size": str(self.quantity),
              "stop_price": str(self.stop_price),
              "limit_price": str(self.limit_price or self.stop_price)
          }
      }
  }
  ```

`_to_kraken(self) -> dict`:
- pair: Convert BTC->XBT, remove slash (e.g., "BTC/USD" -> "XBTUSD")
- For market: `{"pair": pair, "type": self.side, "ordertype": "market", "volume": str(self.quantity)}`
- For limit: `{"pair": pair, "type": self.side, "ordertype": "limit", "price": str(self.limit_price), "volume": str(self.quantity)}`
- For stop: `{"pair": pair, "type": self.side, "ordertype": "stop-loss-limit", "price": str(self.stop_price), "price2": str(self.limit_price or self.stop_price), "volume": str(self.quantity)}`

`@classmethod from_signal(cls, signal: dict) -> "CanonicalOrder"`:
- Accept a dict (from cmc_signals row or in-memory signal output)
- Map keys: `pair` (required), `side` or `direction` (required), `order_type` (default "market"), `quantity` or `size` (required), `limit_price`, `stop_price`, `signal_id`, `asset_id`
- Normalize side: "Long"/"long"/"BUY"/"buy" -> "buy", "Short"/"short"/"SELL"/"sell" -> "sell"
- If both `side` and `direction` keys are present, prefer `side`
- Return CanonicalOrder instance
- Raise ValueError if required keys missing

`__repr__` should show: `CanonicalOrder(BTC/USD buy market 0.01)`

**Class PaperOrderLogger:**

`__init__(self, db_url: str = None)`:
- If db_url is None, resolve via `resolve_db_url()`
- Store db_url
- Create engine with NullPool (matching project pattern)

`log_order(self, order: "CanonicalOrder", exchange: str, environment: str = "sandbox") -> str`:
- Call `order.validate()` first
- Generate exchange_payload via `order.to_exchange(exchange)`
- Serialize payload: `json.dumps(exchange_payload)`
- INSERT into paper_orders table using parameterized query:
  ```sql
  INSERT INTO paper_orders (signal_id, asset_id, exchange, pair, side, order_type,
                            quantity, limit_price, stop_price, exchange_payload,
                            status, environment, client_order_id)
  VALUES (:signal_id, :asset_id, :exchange, :pair, :side, :order_type,
          :quantity, :limit_price, :stop_price, :exchange_payload,
          'paper', :environment, :client_order_id)
  RETURNING order_uuid
  ```
- Return the order_uuid as string
- Log at INFO level: "Paper order logged: {pair} {side} {order_type} {quantity} on {exchange}"

`log_orders_batch(self, orders: List["CanonicalOrder"], exchange: str, environment: str = "sandbox") -> List[str]`:
- Loop over orders, call `log_order()` for each
- Return list of order_uuids
- Log summary: "Logged {n} paper orders on {exchange}"

`get_recent_orders(self, exchange: str = None, limit: int = 20) -> list`:
- SELECT from paper_orders ordered by created_at DESC, with optional exchange filter
- Return list of dicts
- Use LIMIT parameter

Use `with engine.connect() as conn:` and `conn.execute(text(...), params)` pattern. Commit after each insert (or use `conn.begin()` context manager).
