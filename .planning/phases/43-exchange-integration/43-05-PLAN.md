This script polls live prices from Coinbase and Kraken, compares to DB bar closes, and writes results to exchange_price_feed.

Structure:
```python
import argparse
```

**Functions:**

`_get_latest_bar_close(conn, asset_symbol: str, tf: str = "1D") -> tuple[Optional[float], Optional[datetime]]`:
- Query: `SELECT close, ts FROM cmc_price_bars_multi_tf WHERE id = (SELECT id FROM dim_assets WHERE symbol = :symbol LIMIT 1) AND tf = :tf ORDER BY ts DESC LIMIT 1`
- Return (close_price, bar_ts) or (None, None) if no data
- Wrap in try/except, return (None, None) on error

`_get_adaptive_threshold(conn, asset_symbol: str) -> float`:
- Query: `SELECT std_ret_30_252 FROM cmc_asset_stats WHERE id = (SELECT id FROM dim_assets WHERE symbol = :symbol LIMIT 1) AND tf = '1D' ORDER BY ts DESC LIMIT 1`
- If result exists and std_ret_30_252 is not None: return `3.0 * float(std_ret_30_252) * 100` (3-sigma as percentage)
- Else: return DEFAULT_THRESHOLD_PCT
- Wrap in try/except, return DEFAULT_THRESHOLD_PCT on error
- Log DEBUG: "Adaptive threshold for {symbol}: {threshold}% (source: {'cmc_asset_stats' or 'default fallback'})"

NOTE: The column name in cmc_asset_stats may vary. The research says "std_ret_30 from cmc_asset_stats". Check the actual column name. If std_ret_30_252 does not exist, try `std_ret_w30` or just fall back to default. Be defensive -- this is a nice-to-have, not a hard requirement.

`_fetch_live_price(exchange_name: str, pair: str) -> dict`:
- Create adapter via `get_exchange(exchange_name)` (no auth needed for ticker)
- Call `adapter.get_ticker(pair)`
- Return dict with keys: exchange, pair, bid, ask, mid, last_price
- bid/ask may not be available from all exchanges -- set to None if not in response
- mid = (bid + ask) / 2 if both available, else last_price
- Wrap in try/except, return None on error (log WARNING)

`_write_feed_row(conn, row: dict)`:
- INSERT into exchange_price_feed with all columns
- Use parameterized query with :named params

`refresh_price_feed(db_url: str, exchanges: list = None, pairs: list = None, dry_run: bool = False)`:
- Main function
- Create engine with NullPool
- For each exchange x pair combination:
  1. Fetch live price via _fetch_live_price
  2. If fetch failed, skip (already logged)
  3. Get latest bar close via _get_latest_bar_close (derive symbol from pair: "BTC/USD" -> "BTC")
  4. Get adaptive threshold
  5. Compute discrepancy_pct: `abs(live_price - bar_close) / bar_close * 100` if bar_close available
  6. exceeds_threshold: discrepancy_pct > threshold_pct
  7. If exceeds_threshold: log WARNING with details
  8. If dry_run: print summary row, skip DB write
  9. Else: write row to exchange_price_feed via _write_feed_row
- Print summary table at end

**CLI (argparse):**
- `--exchanges` or `-e`: space-separated exchange names (default: coinbase kraken)
- `--pairs` or `-p`: space-separated pairs (default: BTC/USD ETH/USD)
- `--dry-run`: print without writing to DB
- `--db-url`: override DB URL

Entry point: `python -m ta_lab2.scripts.exchange.refresh_exchange_price_feed`

Log format: match project convention (INFO level by default, DEBUG for adaptive threshold details).
