**HMAC-SHA512 signing method `_sign(self, urlpath: str, data: dict) -> str`:**
```python
import urllib.parse, hashlib, hmac, base64

def _sign(self, urlpath: str, data: dict) -> str:
    encoded = (str(data["nonce"]) + urllib.parse.urlencode(data)).encode()
    message = urlpath.encode() + hashlib.sha256(encoded).digest()
    mac = hmac.new(base64.b64decode(self.api_secret), message, hashlib.sha512)
    return base64.b64encode(mac.digest()).decode()
```

**Helper `_private_post(self, endpoint: str, data: dict = None) -> dict`:**
- Call `_requires_auth()` first
- If data is None, data = {}
- Set `data["nonce"] = str(int(time.time() * 1000))` (millisecond resolution)
- Build urlpath: `f"/0/private/{endpoint}"`
- Compute signature via `_sign(urlpath, data)`
- Headers: `{"API-Key": self.api_key, "API-Sign": signature}`
- POST to `f"https://api.kraken.com{urlpath}"` with headers and data
- Call `response.raise_for_status()`
- Parse JSON response
- If `result.get("error")` is non-empty list, raise `InvalidRequestError(f"Kraken error: {result['error']}")`
- Return `result["result"]`

**Helper `_requires_auth(self)` -- raises AuthenticationError if api_key or api_secret is empty/None.**

**Implement authenticated methods:**

`get_account_balances(self) -> Dict[str, float]`:
- Call `_private_post("Balance")`
- Return `{asset: float(balance) for asset, balance in result.items() if float(balance) > 0}`

`get_open_orders(self, pair: str = None) -> List[dict]`:
- Call `_private_post("OpenOrders")`
- Parse the result["open"] dict (keyed by txid)
- For each order, extract: order_id (txid), pair, side (type), ordertype, volume, price, status
- If `pair` is provided, filter by `_normalize_pair(pair)`
- Return list of order dicts

`place_market_order(self, pair: str, side: str, amount: float) -> Dict[str, str]`:
- data = {"pair": self._normalize_pair(pair), "type": side.lower(), "ordertype": "market", "volume": str(amount)}
- Call `_private_post("AddOrder", data)`
- Return `{"order_id": result["txid"][0], "status": "submitted"}`

`place_limit_order(self, pair: str, side: str, price: float, amount: float) -> Dict[str, str]`:
- data = {"pair": self._normalize_pair(pair), "type": side.lower(), "ordertype": "limit", "price": str(price), "volume": str(amount)}
- Call `_private_post("AddOrder", data)`
- Return `{"order_id": result["txid"][0], "status": "submitted"}`

`cancel_order(self, order_id: str) -> Dict[str, str]`:
- data = {"txid": order_id}
- Call `_private_post("CancelOrder", data)`
- Return `{"order_id": order_id, "status": "cancelled"}`
