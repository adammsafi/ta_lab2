**drift_metrics.py:**

1. `DriftMetrics` frozen dataclass with fields (matching cmc_drift_metrics columns):
   - metric_date: date
   - config_id: int
   - asset_id: int
   - signal_type: str
   - pit_replay_run_id: str | None (UUID as string)
   - cur_replay_run_id: str | None
   - paper_trade_count: int
   - replay_trade_count: int
   - unmatched_paper: int
   - unmatched_replay: int
   - paper_cumulative_pnl: float | None
   - replay_pit_cumulative_pnl: float | None
   - replay_cur_cumulative_pnl: float | None
   - absolute_pnl_diff: float | None
   - data_revision_pnl_diff: float | None
   - tracking_error_5d: float | None
   - tracking_error_30d: float | None
   - paper_sharpe: float | None
   - replay_sharpe: float | None
   - sharpe_divergence: float | None
   - threshold_breach: bool = False
   - drift_pct_of_threshold: float | None = None

2. `compute_rolling_tracking_error(paper_daily_pnl: np.ndarray, replay_daily_pnl: np.ndarray, window: int = 5) -> np.ndarray`:
   - Computes rolling std of (paper - replay) daily P&L differences.
   - Uses pd.Series.rolling(window=window, min_periods=window).std().values
   - Returns array of same length as input, with NaN for first (window-1) entries.
   - When input arrays have fewer than `window` elements, returns all NaN.

3. `compute_sharpe(daily_pnl: np.ndarray) -> float | None`:
   - Returns annualized Sharpe ratio: mean / std * sqrt(365)
   - Returns None if std == 0 or len < 2.

4. `compute_drift_metrics(config_id: int, asset_id: int, signal_type: str, metric_date: date, paper_daily_pnl: np.ndarray, replay_pit_daily_pnl: np.ndarray, replay_cur_daily_pnl: np.ndarray, paper_trade_count: int, replay_trade_count: int, unmatched_paper: int, unmatched_replay: int, pit_replay_run_id: str | None, cur_replay_run_id: str | None, threshold: float = 0.015, window: int = 5) -> DriftMetrics`:
   - Computes cumulative P&L (sum of daily arrays).
   - Computes absolute_pnl_diff = abs(paper_cumulative - replay_pit_cumulative).
   - Computes data_revision_pnl_diff = abs(replay_pit_cumulative - replay_cur_cumulative).
   - Computes tracking_error_5d = last non-NaN value from compute_rolling_tracking_error(paper, replay_pit, 5).
   - Computes tracking_error_30d = last non-NaN value from compute_rolling_tracking_error(paper, replay_pit, 30).
   - Computes paper_sharpe and replay_sharpe via compute_sharpe.
   - sharpe_divergence = abs(paper_sharpe - replay_sharpe) if both non-None, else None.
   - threshold_breach = tracking_error_5d is not None and tracking_error_5d > threshold.
   - drift_pct_of_threshold = (tracking_error_5d / threshold * 100) if tracking_error_5d is not None else None.
   - Returns DriftMetrics with all fields populated.

**data_snapshot.py:**

1. `collect_data_snapshot(conn, asset_ids: list[int]) -> dict`:
   - For each asset_id, query MAX(ts) from cmc_price_bars_multi_tf WHERE id=:asset_id AND tf='1D', similarly for cmc_features and cmc_ema_multi_tf_u.
   - Return dict keyed by str(asset_id), values are dicts with latest_bar_ts, latest_feature_ts, ema_latest_ts (all as ISO strings or None).
   - Uses sqlalchemy text() for queries. Import from sqlalchemy.
