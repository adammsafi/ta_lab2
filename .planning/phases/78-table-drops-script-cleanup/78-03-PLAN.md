<objective>
Validate row count parity across all 6 families, drop all 30 siloed data tables, run VACUUM FULL, and capture storage reduction metrics.

Purpose: This is the core destructive operation of Phase 78 -- reclaiming ~207 GB (48% of DB) by dropping tables that are fully superseded by the _u unified tables. Pre-drop validation ensures no data loss. State tables are intentionally preserved (see design note).

Output: 30 tables dropped, ~207 GB reclaimed, before/after metrics captured in summary.

**DESIGN NOTE -- State Tables:**
The CONTEXT.md says "drop state tables too" but research (78-RESEARCH.md) confirmed all 30 siloed-path state tables are STILL ACTIVELY USED by builder scripts for watermark tracking (STATE_TABLE class constants). Dropping them would break every incremental refresh. State tables total only 78 MB (0.04% of the 207 GB being reclaimed). Decision: KEEP state tables. Renaming/migrating them is a follow-up concern, not Phase 78 scope.
</objective>

Run the following Python script to machine-check parity results. This script MUST exit 0 (all families pass) before proceeding to Task 2. If it exits non-zero, STOP -- do NOT proceed to drops.

```python
python -c "
import os, sys
from sqlalchemy import create_engine, text

db_url = os.environ.get('TARGET_DB_URL', 'postgresql://localhost/marketdata')
engine = create_engine(db_url)

families = ['price_bars', 'returns_bars', 'ema', 'returns_ema', 'ama', 'returns_ama']
base_tables = {
    'price_bars': 'price_bars_multi_tf',
    'returns_bars': 'returns_bars_multi_tf',
    'ema': 'ema_multi_tf',
    'returns_ema': 'returns_ema_multi_tf',
    'ama': 'ama_multi_tf',
    'returns_ama': 'returns_ama_multi_tf',
}
suffixes = ['', '_cal_us', '_cal_iso', '_cal_anchor_us', '_cal_anchor_iso']
failures = []

with engine.connect() as conn:
    for fam in families:
        base = base_tables[fam]
        siloed_total = 0
        for suf in suffixes:
            row = conn.execute(text(f'SELECT COUNT(*) FROM public.{base}{suf}')).fetchone()
            siloed_total += row[0]
        u_row = conn.execute(text(f'SELECT COUNT(*) FROM public.{base}_u')).fetchone()
        u_total = u_row[0]
        status = 'PASS' if u_total >= siloed_total else 'FAIL'
        print(f'{fam}: siloed={siloed_total:,}  u={u_total:,}  {status}')
        if u_total < siloed_total:
            failures.append(f'{fam}: u_total({u_total}) < siloed_total({siloed_total})')

if failures:
    print(f'\nABORT: {len(failures)} family(ies) failed parity:')
    for f in failures:
        print(f'  - {f}')
    sys.exit(1)
else:
    print('\nAll 6 families PASS parity check. Safe to proceed with drops.')
    sys.exit(0)
"
```

Exit code 0 = all families pass, safe to proceed. Exit code 1 = parity failure, STOP.
