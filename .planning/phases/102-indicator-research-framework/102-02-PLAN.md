**haircut_sharpe(sr_observed, n_trials, n_obs, freq='monthly') -> dict**

Implementation (Bonferroni method per Harvey & Liu 2015):
1. Convert SR to monthly: sr_m = sr_observed / sqrt(12)
2. Frequency scaling: FREQ_TO_MONTHLY = {'daily': 21.0, 'weekly': 4.33, 'monthly': 1.0, 'annual': 1/12}. Compute n_monthly = n_obs / scale.
3. Compute t-stat: t = sr_m * sqrt(n_monthly)
4. One-sided p-value: p = 1 - norm.cdf(t)
5. Bonferroni: p_adj = min(1.0, p * n_trials)
6. Adjusted t: t_adj = norm.ppf(1 - p_adj) if p_adj < 1.0 else 0.0
7. Back to SR: sr_haircut = (t_adj / sqrt(n_monthly)) * sqrt(12)
8. Haircut %: (sr_observed - sr_haircut) / sr_observed if sr_observed > 0 else 0.0

Return dict: {sr_observed, sr_haircut, haircut_pct, n_trials, n_obs, freq}

Edge cases:
- sr_observed <= 0: return sr_haircut=0.0, haircut_pct=0.0 (negative Sharpe already bad)
- n_trials <= 0: return sr_haircut=sr_observed, haircut_pct=0.0 (no penalty)
- n_obs <= 0: return sr_haircut=0.0, haircut_pct=1.0

Also add a helper: **get_trial_count(conn) -> int** that runs `SELECT COUNT(*) FROM trial_registry` and returns the total trial count. This is how haircut_sharpe gets its n_trials input.

**haircut_ic_ir(conn, ic_ir_observed, n_obs, indicator_name=None, tf=None) -> dict**

This applies the SAME Bonferroni HL method but to IC-IR instead of Sharpe:
1. Get n_trials via get_trial_count(conn)
2. Treat IC-IR like a t-statistic scaled ratio (same mathematical structure as Sharpe):
   - t = ic_ir_observed * sqrt(n_obs)
   - p = 1 - norm.cdf(t)
   - p_adj = min(1.0, p * n_trials)
   - t_adj = norm.ppf(1 - p_adj) if p_adj < 1.0 else 0.0
   - ic_ir_haircut = t_adj / sqrt(n_obs)
   - haircut_pct = (ic_ir_observed - ic_ir_haircut) / ic_ir_observed if ic_ir_observed > 0 else 0.0
3. If indicator_name AND tf are provided, write ic_ir_haircut back to trial_registry:
   `UPDATE trial_registry SET haircut_ic_ir = %s WHERE indicator_name = %s AND tf = %s`
4. Return dict: {ic_ir_observed, ic_ir_haircut, haircut_pct, n_trials, n_obs}

Edge cases: same as haircut_sharpe (negative/zero IC-IR, zero trials, zero obs).

Use scipy.stats.norm for CDF/PPF. Import at top of file.

**block_bootstrap_ic(feature, fwd_returns, n_boot=1000, seed=42) -> dict**

Implementation:
1. Drop NaN pairs from both arrays (same as permutation_ic_test)
2. Compute observed IC via spearmanr
3. Compute adaptive block length: `opt = optimal_block_length(ret_clean)`, then `block_len = max(1, int(np.ceil(float(opt["stationary"].iloc[0]))))`.
   **CRITICAL**: Use column name `"stationary"`, NOT `"b_sb"` (arch 8.0.0 naming).
4. Create StationaryBootstrap: `bs = StationaryBootstrap(block_len, ret_clean, seed=seed)`
5. For each bootstrap sample, compute IC between feature and bootstrapped returns. Only include samples where len(boot_ret) == len(feat_clean).
6. Compute 2.5th and 97.5th percentiles of bootstrap IC distribution.
7. Return dict: {ic_obs, ci_lo, ci_hi, block_len, n_boot (actual count of valid bootstrap samples), n_obs}

Edge cases:
- Fewer than 20 valid observations: return ci_lo=NaN, ci_hi=NaN, block_len=0
- optimal_block_length returns NaN or 0: use block_len=1 as fallback

Imports at top of file:
```python
from arch.bootstrap import StationaryBootstrap, optimal_block_length
```

Add a note in the module docstring about the arch 8.0.0 column name change.
  </action>

```python
import numpy as np
from ta_lab2.analysis.multiple_testing import block_bootstrap_ic

# --- Test 1: Basic functionality with strongly correlated series ---
rng = np.random.default_rng(42)
x = np.arange(200, dtype=float)
y = x + rng.normal(0, 20, 200)
result = block_bootstrap_ic(x, y, n_boot=500)
assert result['ci_lo'] > 0  # CI should not cross zero for strong signal
assert result['ci_hi'] > result['ci_lo']
assert result['block_len'] >= 1
print(f"IC={result['ic_obs']:.3f}, CI=[{result['ci_lo']:.3f}, {result['ci_hi']:.3f}], block={result['block_len']}")

# --- Test 2: Block bootstrap CI must be wider than naive IID bootstrap ---
# Generate autocorrelated (AR(1)) series where block bootstrap matters
rng2 = np.random.default_rng(99)
n = 300
phi = 0.7  # strong autocorrelation
noise = rng2.normal(0, 1, n)
ar1 = np.zeros(n)
ar1[0] = noise[0]
for i in range(1, n):
    ar1[i] = phi * ar1[i-1] + noise[i]
feat_ar = np.arange(n, dtype=float)

# Block bootstrap CI
bb_result = block_bootstrap_ic(feat_ar, ar1, n_boot=500, seed=123)
bb_width = bb_result['ci_hi'] - bb_result['ci_lo']

# Naive IID bootstrap CI (resample individual observations with replacement)
from scipy.stats import spearmanr
boot_ics = []
for _ in range(500):
    idx = rng2.choice(n, size=n, replace=True)
    ic_boot, _ = spearmanr(feat_ar[idx], ar1[idx])
    boot_ics.append(ic_boot)
naive_lo = np.percentile(boot_ics, 2.5)
naive_hi = np.percentile(boot_ics, 97.5)
naive_width = naive_hi - naive_lo

print(f"Block bootstrap CI width: {bb_width:.4f}")
print(f"Naive IID bootstrap CI width: {naive_width:.4f}")
assert bb_width > naive_width, (
    f"Block bootstrap CI ({bb_width:.4f}) must be wider than naive IID CI ({naive_width:.4f}) "
    f"for autocorrelated AR(1) series with phi={phi}"
)
```
