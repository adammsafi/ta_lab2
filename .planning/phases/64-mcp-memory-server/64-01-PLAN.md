Create `mcp_server.py` with 6 MCP tools using `fastmcp>=3.0.0` (standalone package, NOT `mcp.server.fastmcp`).

Create FastMCP instance:
```python
mcp = FastMCP(
    name="ta-lab2-memory",
    instructions="Memory server for the ta_lab2 trading system project. ..."
)
```

Define 6 tools using `@mcp.tool` decorator with `Annotated` type hints for auto-schema generation:

1. **memory_search(query, top_k=10, min_similarity=0.6, category=None)** -> dict
   - Uses `get_mem0_client().search(query=query, limit=top_k)`
   - Filters results by min_similarity threshold
   - Returns `{"query": str, "results": [...], "count": int}` where each result has id, content, metadata, similarity
   - CRITICAL: Mem0 search returns either a list or `{"results": [...]}` dict. Handle both formats.
   - Results from Mem0 have keys: `id`, `memory` (not `content`), `metadata`, `score`

2. **memory_context(query, max_memories=10, min_similarity=0.6)** -> dict
   - CRITICAL: Do NOT import or call `inject_memory_context()` from `injection.py`. That function internally calls `search_memories()` from `query.py` which uses ChromaDB. ChromaDB is NOT available in Docker (only Qdrant via Mem0).
   - Instead, implement a local helper approach:
     (a) Call `get_mem0_client().search(query=query, limit=max_memories)` to get Mem0 results
     (b) Convert each Mem0 result dict to a `SearchResult` dataclass instance (imported from `query.py`):
         ```python
         from .query import SearchResult
         # For each Mem0 result:
         SearchResult(
             memory_id=result["id"],
             content=result["memory"],  # Mem0 uses "memory" key, not "content"
             metadata=result.get("metadata", {}),
             similarity=result.get("score", 0.0),
             distance=1.0 - result.get("score", 0.0),
         )
         ```
     (c) Filter by min_similarity threshold
     (d) Call `format_memories_for_prompt(results=adapted_results)` from `injection.py` — this function has NO ChromaDB dependency, it only formats SearchResult objects into markdown
   - Returns `{"query": str, "context": str}` where context is formatted markdown
   - NOTE: Importing `SearchResult` from `query.py` and `format_memories_for_prompt` from `injection.py` is safe — `SearchResult` is a plain dataclass with no ChromaDB import at class-definition time, and `format_memories_for_prompt` only uses the `SearchResult` fields. The ChromaDB dependency is only in `search_memories()` function body (lazy import of `client.py`), which we never call.

3. **memory_store(content, source="claude_code", category=None, user_id="orchestrator")** -> dict
   - Uses `get_mem0_client().add(messages=[...], user_id=user_id, metadata={source, category}, infer=True)`
   - Returns the Mem0 add result (operation type: ADD/UPDATE/DELETE/NOOP)

4. **memory_stats()** -> dict
   - Uses `get_mem0_client().memory_count` property
   - Returns `{"total_memories": int, "collection_name": "project_memories", "backend": "qdrant"}`

5. **memory_health(staleness_days=90)** -> dict
   - Uses `MemoryHealthMonitor(staleness_days=staleness_days).generate_health_report()`
   - Returns report fields: total_memories, healthy, stale, deprecated, missing_metadata, age_distribution

6. **list_categories()** -> dict
   - Uses `get_mem0_client().get_all(user_id="orchestrator")`
   - Extracts unique category values from metadata
   - Returns `{"categories": sorted list, "count": int}`

Important patterns:
- Use `from typing import Annotated, Optional` and `from pydantic import Field` for parameter annotations
- All tools are sync functions (FastMCP runs them in threadpool automatically)
- Use lazy imports inside tool functions to avoid import errors at module load time
- Export `mcp` at module level for server.py to import
- Add `__all__ = ["mcp"]`

**server.py** — Create combined ASGI application that mounts MCP + REST on same port:

```python
from fastapi import FastAPI
from .mcp_server import mcp
from .api import create_memory_api

def create_app() -> FastAPI:
    """Create combined ASGI app with MCP + REST."""
    # Create MCP ASGI app (mounted at root of its sub-path)
    mcp_app = mcp.http_app(path="/")

    # Create FastAPI app WITH MCP lifespan (CRITICAL for session management)
    api = create_memory_api(lifespan=mcp_app.lifespan)

    # Mount MCP endpoint at /mcp/
    api.mount("/mcp", mcp_app)

    return api

app = create_app()

# Run: uvicorn ta_lab2.tools.ai_orchestrator.memory.server:app --host 0.0.0.0 --port 8080
```

Add `__all__ = ["create_app", "app"]`.
