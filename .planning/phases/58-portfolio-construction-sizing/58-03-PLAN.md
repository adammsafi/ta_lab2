  <action>
Create `src/ta_lab2/portfolio/black_litterman.py` with a `BLAllocationBuilder` class.

**Constructor:**
- Accepts optional `config: dict = None`. If None, calls `load_portfolio_config()`.
- Extracts BL params: tau (0.05), use_idzorek (True), min_view_confidence (0.2),
  max_view_confidence (0.8), views_source ('all_active').
- Extracts optimizer params for weight_bounds: max_position_pct.

**Method: `signals_to_mu(signal_scores, ic_ir, base_vol) -> pd.Series`**
- `signal_scores`: pd.DataFrame indexed by asset_id, columns are signal types, values are scores.
- `ic_ir`: pd.Series indexed by signal_type with rolling IC-IR values.
- `base_vol`: pd.Series indexed by asset_id with annualized volatility.
- Implementation from 58-RESEARCH.md Pattern 4:
  1. Clip IC-IR to >= 0 (negative IC-IR signals are noise).
  2. Normalize to sum to 1.0 (these are weights per signal type).
  3. Compute weighted composite score per asset: `signal_scores.mul(weights, axis='columns').sum(axis=1)`.
  4. Z-score cross-sectionally.
  5. Scale to return space: `z * base_vol * 0.1` (10% of vol as max expected alpha).
- Return pd.Series of expected returns indexed by asset_id.

**Method: `build_views(signal_scores, ic_ir) -> tuple[dict, list]`**
- Convert signal_scores to absolute views dict: {asset_id: expected_return}.
- Compute view_confidences: normalize IC-IR to [min_view_confidence, max_view_confidence] range.
  `conf = min_conf + (ic_ir_normalized * (max_conf - min_conf))` where ic_ir_normalized
  scales the raw IC-IR values to [0, 1] via min-max normalization within the set.
- Filter views where IC-IR > 0.1 threshold (views_source: all_active means use all that
  pass the threshold).
- Return (absolute_views_dict, view_confidences_list).

**Method: `run(prices, market_caps, signal_scores, ic_ir, base_vol, S=None) -> dict`**
- `prices`: pd.DataFrame (DatetimeIndex x asset columns) of close prices.
- `market_caps`: pd.Series (asset -> latest market cap). Must be fetched at same timestamp
  for all assets (Pitfall 5 from research).
- `signal_scores`, `ic_ir`, `base_vol` as above.
- `S`: optional pre-computed covariance (from optimizer.run_all). If None, compute via
  `CovarianceShrinkage(prices).ledoit_wolf()`.

Steps:
1. Compute `delta = black_litterman.market_implied_risk_aversion(prices)`.
2. Compute `prior = black_litterman.market_implied_prior_returns(market_caps, delta, S)`.
3. Build views: `absolute_views, view_confidences = self.build_views(signal_scores, ic_ir)`.
   - If no views pass the IC-IR threshold, log warning and return prior-only weights
     (optimize on prior returns directly).
4. Create `BlackLittermanModel(S, pi=prior, absolute_views=absolute_views, omega="idzorek",
   view_confidences=view_confidences, tau=self.tau)`.
5. Get posterior: `bl_returns = bl.bl_returns()`, `bl_cov = bl.bl_cov()`.
6. Optimize on posterior: `EfficientFrontier(bl_returns, bl_cov, weight_bounds=(0, max_pos))`.
   Try `max_sharpe()`, fall back to `min_volatility()` on a fresh EF instance if it fails
   (same pattern as optimizer.py).
7. Return dict:
   ```python
   {"bl": ef.clean_weights(), "bl_returns": bl_returns, "bl_cov": bl_cov,
    "prior": prior, "views": absolute_views, "view_confidences": view_confidences}
   ```

All logging via `logging.getLogger(__name__)`.

1. Create `src/ta_lab2/portfolio/bet_sizing.py` with:

**Standalone function: `probability_bet_size(signal_probability, side, w=2.0) -> float`**
- From 58-RESEARCH.md Pattern 5 (de Prado Chapter 10):
  ```python
  z = (signal_probability - 0.5) * w
  return side * (2 * norm.cdf(z) - 1)
  ```
- `signal_probability`: float in [0, 1], P(signal is correct).
- `side`: +1 for long, -1 for short.
- `w`: width parameter (larger = more aggressive). Default 2.0.
- Returns float in [-1, 1]. At prob=0.5 -> 0 (no edge). At prob=1.0 -> approaches 1.

**Class: `BetSizer`**
- Constructor accepts optional config dict. Extracts bet_sizing section: mode, w_parameter, min_confidence.
- **Method: `scale_weights(raw_weights, signal_probabilities, sides) -> dict`**
  - `raw_weights`: dict[asset_id, float] from optimizer.
  - `signal_probabilities`: dict[asset_id, float] P(signal correct) per asset.
  - `sides`: dict[asset_id, int] (+1 or -1) per asset.
  - For mode='optimizer_first': `final_weight = raw_weight * probability_bet_size(prob, side, w)`.
  - For mode='sizing_as_constraints': Return `{asset: (lower_bound, upper_bound)}` dict where
    bounds are derived from bet size (to be fed as weight_bounds to optimizer).
  - Skip assets where prob < min_confidence (set final_weight to 0).
  - Return dict[asset_id, float] of scaled weights.

- **Method: `compute_bounds(signal_probabilities, sides, max_position_pct) -> dict`**
  - For sizing_as_constraints mode. Returns dict[asset_id, tuple[float, float]] of
    (lower, upper) weight bounds per asset based on bet sizing.
  - Upper bound = max_position_pct * abs(bet_size). Lower bound = 0 for long-only.
