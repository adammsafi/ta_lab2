Create `src/ta_lab2/portfolio/optimizer.py` with a `PortfolioOptimizer` class.

**Constructor:**
- Accepts optional `config: dict = None`. If None, calls `load_portfolio_config()`.
- Extracts optimizer params: lookback_calendar_days, min_lookback_bars, max_position_pct, max_gross_exposure, cvar_beta, condition_number_threshold.
- Extracts regime_routing dict.
- Stores config for snapshot persistence.

**Core method: `run_all(prices: pd.DataFrame, regime_label: str | None = None, tf: str = "1D") -> dict`**
- `prices` is a DataFrame with DatetimeIndex rows and asset columns (tickers or integer IDs), values are close prices.
- **Adaptive lookback:** Resolve `tf_days_nominal` from `DimTimeframe().tf_days(tf)` (from `ta_lab2.time.dim_timeframe`).
  Compute `lookback_bars = round(lookback_calendar_days / tf_days_nominal)`.
  If `lookback_bars < min_lookback_bars`, raise ValueError: f"Lookback {lookback_bars} bars (from {lookback_calendar_days} cal days / {tf_days_nominal}d per bar) is below minimum {min_lookback_bars} for tf={tf}".
  Slice prices to last `lookback_bars` rows: `prices_window = prices.tail(lookback_bars)`.
- Compute expected returns via `expected_returns.ema_historical_return(prices_window, span=lookback_bars)`.
- Compute covariance via `risk_models.CovarianceShrinkage(prices_window).ledoit_wolf()`.
- Check condition number via `np.linalg.cond(S.values)`.
- Run all three optimizers (fresh EfficientFrontier instance for each -- CRITICAL to avoid state contamination):

  **Mean-Variance (mv):**
  - `EfficientFrontier(mu, S, weight_bounds=(0, max_position_pct))`
  - Try `ef.max_sharpe()` first. If OptimizationError (infeasible, e.g. all-negative returns), fall back to `ef.min_volatility()` on a FRESH instance. If both fail, set weights_mv = None.
  - `ef.clean_weights()` returns the weight dict.

  **CVaR (cvar):**
  - Compute `returns_df = expected_returns.returns_from_prices(prices)`.
  - `EfficientCVaR(mu, returns_df, beta=cvar_beta, weight_bounds=(0, max_position_pct))`
  - Call `ef_cvar.min_cvar()`. If fails, weights_cvar = None.
  - `ef_cvar.clean_weights()`.

  **HRP (hrp):**
  - Filter both returns_df and S to common_assets (intersection of columns) to avoid shape mismatch (Pitfall 6).
  - `HRPOpt(returns=returns_df[common], cov_matrix=S.loc[common, common])`
  - `hrp.optimize(linkage_method="ward")`, then `hrp.clean_weights()`.
  - HRP should never fail (no matrix inversion), but wrap in try/except as safety.

- **Return dict:**
  ```python
  {
      "mv": weights_mv,       # dict[asset, float] or None
      "cvar": weights_cvar,   # dict[asset, float] or None
      "hrp": weights_hrp,     # dict[asset, float] or None
      "active": active_name,  # str: which optimizer is "active"
      "condition_number": float(cond),
      "ill_conditioned": bool,
      "mu": mu,               # pd.Series
      "S": S,                 # pd.DataFrame (cov matrix)
  }
  ```

**Regime routing method: `_select_active(regime_label, ill_conditioned) -> str`**
- If ill_conditioned (condition_number > threshold): return "hrp" (auto-fallback, log warning).
- Else: look up regime_label in regime_routing dict. Default to config["regime_routing"]["default"] if label not found.
- If the selected optimizer returned None weights (failed), fall back to "hrp".

**Helper: `is_ill_conditioned(S, threshold) -> bool`**
- `np.linalg.cond(S.values) > threshold`

**Convenience method: `get_active_weights(result: dict) -> dict[str, float]`**
- Returns `result[result["active"]]` — the weight dict for whichever optimizer is active.
- Returns empty dict if active optimizer's weights are None.
