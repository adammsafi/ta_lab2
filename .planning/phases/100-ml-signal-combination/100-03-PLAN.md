**2. Create `src/ta_lab2/ml/meta_filter.py`** with class `MetaLabelFilter`:

**Constructor `__init__(self, engine)`:**
- Stores engine reference.

**Method `load_training_data(tf="1D", venue_id=1)`:**
- Query `triple_barrier_labels` for all assets at given tf.
- Join with `features` table on (asset_id, tf, t0=ts) to get point-in-time feature values at the trade entry time t0.
- Build binary target: y = 1 if bin > 0 (profit target hit), y = 0 otherwise (stop or timeout).
- Note: triple_barrier_labels has no `primary_side` column. The binary target is simply "did this trade hit profit target?" which is model-agnostic.
- Filter to only features in `dim_feature_selection` (tier='active') + CTF features.
- Drop rows with NaN in features or target.
- Return X (features DataFrame), y (binary Series), t1_series (label end timestamps for purged CV).

**Method `train(X, y, t1_series, n_splits=5, embargo_frac=0.01)`:**
- Compute class imbalance: `neg = (y==0).sum(); pos = (y==1).sum(); scale_pos_weight = neg/pos`.
- Use `PurgedKFoldSplitter(n_splits=n_splits, t1_series=t1_series, embargo_frac=embargo_frac)`.
- For each fold:
  a. Fit `xgb.XGBClassifier(n_estimators=200, max_depth=4, scale_pos_weight=scale_pos_weight, eval_metric='logloss', verbosity=0)`.
  b. Predict probabilities on test set.
  c. Compute accuracy, precision, recall, F1, AUC-ROC.
- After CV, train final model on all data. Store as `self.model_`.
- Return dict with per-fold metrics and means.
- Do NOT include `use_label_encoder=False` (removed in XGBoost 2.0+).

**Method `save_model(path)`:**
- `self.model_.save_model(path)` (XGBoost native save).

**Method `load_model(path)`:**
- `self.model_ = xgb.XGBClassifier(); self.model_.load_model(path)`.

**Method `predict_confidence(X)`:**
- Returns `self.model_.predict_proba(X)[:, 1]` -- P(class=1) = P(trade success).

**Method `evaluate_threshold_impact(X_test, y_test, thresholds=[0.3, 0.4, 0.5, 0.6, 0.7])`:**
- For each threshold: compute how many trades pass (P >= threshold), accuracy among passed trades, and percentage of profitable trades captured.
- Return DataFrame with columns: threshold, n_trades_total, n_trades_passed, pass_rate, accuracy_passed, profitable_capture_rate.
- This is the "backtest impact measurement" for ML-03.

**Method `log_results(engine, cv_results, feature_names, threshold)`:**
- Use `ExperimentTracker(engine).log_run()` with:
  - run_name="xgb_meta_filter_v1"
  - model_type="xgb_meta_filter"
  - cv_method="purged_kfold"
  - oos_accuracy=mean_accuracy
  - oos_sharpe=mean_auc (AUC-ROC as quality metric)
  - feature_set=feature_names
  - notes=f"threshold={threshold}, mean_f1={mean_f1:.4f}, precision={mean_precision:.4f}"

**Lazy-import xgboost** inside methods. Informative ImportError if not installed.

In `_process_asset_signal()`, add the meta-label filter check AFTER the price/portfolio lookups and BEFORE `PositionSizer.compute_target_position()` (approximately line 517 area). The insertion point is after line ~515 (garch_vol block) and before line ~518 (target_qty computation):

```python
# --- meta-label confidence filter ---
if getattr(config, "meta_filter_enabled", False) and self._meta_filter is not None:
    try:
        # Build feature vector for this asset at this signal's timestamp
        features = self._load_meta_features(conn, asset_id, signal)
        if features is not None:
            confidence = self._meta_filter.predict_confidence(features)[0]
            threshold = getattr(config, "meta_filter_threshold", 0.5)
            if confidence < threshold:
                self.logger.info(
                    "_process_asset_signal: meta-filter SKIP asset_id=%d "
                    "confidence=%.4f < threshold=%.4f",
                    asset_id, confidence, threshold,
                )
                return {"skipped_meta_filter": True}
            self.logger.info(
                "_process_asset_signal: meta-filter PASS asset_id=%d "
                "confidence=%.4f >= threshold=%.4f",
                asset_id, confidence, threshold,
            )
    except Exception as exc:
        self.logger.warning(
            "_process_asset_signal: meta-filter error for asset_id=%d: %s, "
            "proceeding without filter",
            asset_id, exc,
        )
```
