  <action>
1. Add `xgboost>=3.2.0` and `shap>=0.51.0` to pyproject.toml dependencies section. Do NOT remove any existing dependencies. Run `pip install -e .` to install.

2. Create `src/ta_lab2/ml/ranker.py` with class `CrossSectionalRanker`:

   **Data loading method `load_features(engine, tf="1D", venue_id=1)`:**
   - Query `features` table for all assets, all timestamps, filtering to active-tier features from `dim_feature_selection` (tier='active') PLUS CTF-promoted features from `dim_feature_selection` WHERE source='ctf_ic_promoted'.
   - Also load AMA features from the features table (columns matching `*_ama` pattern).
   - Join with `returns_bars_multi_tf` to get forward 1D return as the ranking target (shifted by -1 so target is next-bar return).
   - Return DataFrame with columns: ts, asset_id, feature columns, forward_return.
   - Use `pd.to_datetime(utc=True)` for timestamps (not parse_dates).

   **Group array builder `_build_group_array(ts_series)`:**
   - `ts_series.value_counts(sort=False).sort_index().values` -- each element is the count of assets in that period.
   - Validate `sum(group) == len(ts_series)`.

   **Ranking target builder `_build_rank_target(df)`:**
   - For each timestamp, rank assets by forward_return using `df.groupby('ts')['forward_return'].rank(pct=True)`.
   - This gives percentile ranks in [0, 1] suitable for LGBMRanker.

   **Cross-validate method `cross_validate(df, n_splits=5, embargo_frac=0.01)`:**
   - Build t1_series: index=t0 timestamps, values=t0+1D (single-period labels).
   - Use `PurgedKFoldSplitter(n_splits=n_splits, t1_series=t1_series, embargo_frac=embargo_frac)`.
   - For each fold:
     a. Split X, y by train/test indices
     b. RECOMPUTE group_train from ts_train using `_build_group_array()` -- critical, group sizes change per fold
     c. Fit `lgb.LGBMRanker(n_estimators=200, num_leaves=31, learning_rate=0.05, verbose=-1)` with X_train, y_train, group=group_train
     d. Predict on X_test, compute per-period Spearman IC: for each test timestamp, `spearmanr(y_true, y_pred)`. Average across periods.
     e. Compute NDCG: for each test timestamp, `ndcg_score([y_true_period], [y_pred_period])`. Average across periods.
   - Return dict with: per-fold IC list, mean_ic, ic_ir (mean/std), per-fold NDCG list, mean_ndcg.
   - Store the last fold's model as `self.model_` for SHAP analysis (Plan 100-02).

   **Train-full method `train_full(df)`:**
   - Train on ALL data (for production use / SHAP). Store as `self.model_`.
   - Return the fitted model.

   **Log method `log_results(engine, cv_results, feature_names)`:**
   - Use `ExperimentTracker(engine).log_run()` with:
     - run_name="lgbm_ranker_ctf_ama_v1"
     - model_type="lgbm_ranker"
     - cv_method="purged_kfold"
     - oos_accuracy=mean_ic (Spearman IC is the primary metric)
     - oos_sharpe=ic_ir (IC information ratio)
     - feature_set=feature_names
     - notes=f"NDCG={mean_ndcg:.4f}, n_folds={n_splits}, n_features={len(feature_names)}"

   **Import pattern:** Lazy-import LightGBM inside methods (same pattern as double_ensemble.py). Raise informative ImportError at call time, not import time.

   **numpy scalar safety:** Use `_to_python()` helper (same as experiment_tracker.py) for any values bound to SQL.
