**Step 0 (pre-flight): Discover actual PK columns for all 5 state tables.**

Write and run a small Python script (do NOT commit) that queries `pg_index` for each state table and prints the PK columns:

```python
import os
from sqlalchemy import create_engine, text

db_url = os.environ.get("TARGET_DB_URL") or open("db_config.env").read().strip().split("=",1)[1]
engine = create_engine(db_url)

STATE_TABLES = [
    "public.price_bars_multi_tf_state",
    "public.price_bars_multi_tf_cal_us_state",
    "public.price_bars_multi_tf_cal_iso_state",
    "public.price_bars_multi_tf_cal_anchor_us_state",
    "public.price_bars_multi_tf_cal_anchor_iso_state",
]

with engine.connect() as conn:
    for tbl in STATE_TABLES:
        rows = conn.execute(text("""
            SELECT a.attname
            FROM pg_index i
            JOIN pg_attribute a ON a.attrelid = i.indrelid AND a.attnum = ANY(i.indkey)
            WHERE i.indrelid = :tbl::regclass AND i.indisprimary
            ORDER BY array_position(i.indkey, a.attnum)
        """), {"tbl": tbl}).fetchall()
        pk_cols = [r[0] for r in rows]
        print(f"{tbl}: PK = ({', '.join(pk_cols)})")
```

Capture and log the output. The PKs will be either `(id, tf)` or `(id, tf, venue_id)`.

**Step 1: Generate the correct bootstrap SQL based on discovered PKs.**

For each state table, use the appropriate ON CONFLICT clause:

- If PK is `(id, tf)`: use `ON CONFLICT (id, tf) DO UPDATE SET ...`
- If PK is `(id, tf, venue_id)`: use `ON CONFLICT (id, tf, venue_id) DO UPDATE SET ...`

**Bootstrap SQL template (adapt ON CONFLICT per table):**

```sql
INSERT INTO {state_table} (id, tf, venue_id, venue, daily_min_seen, daily_max_seen, last_bar_seq, last_time_close, updated_at)
SELECT
    u.id, u.tf, u.venue_id,
    COALESCE(dv.venue, 'CMC_AGG') AS venue,
    MIN(u.timestamp), MAX(u.timestamp), MAX(u.bar_seq), MAX(u.timestamp), NOW()
FROM public.price_bars_multi_tf_u u
LEFT JOIN public.dim_venues dv ON dv.venue_id = u.venue_id
WHERE u.alignment_source = :alignment_source
GROUP BY u.id, u.tf, u.venue_id, dv.venue
ON CONFLICT ({pk_cols_csv}) DO UPDATE SET
    last_time_close = GREATEST({state_table_name}.last_time_close, EXCLUDED.last_time_close),
    last_bar_seq = GREATEST({state_table_name}.last_bar_seq, EXCLUDED.last_bar_seq),
    daily_max_seen = GREATEST({state_table_name}.daily_max_seen, EXCLUDED.daily_max_seen),
    updated_at = NOW();
```
