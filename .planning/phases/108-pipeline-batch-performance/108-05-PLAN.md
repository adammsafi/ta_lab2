1. **Capture pre-run checksums** for all returns tables (to compare after pipeline run):
   ```sql
   -- For each returns table:
   SELECT
     'returns_ema_multi_tf_u' as tbl,
     COUNT(*) as row_count,
     SUM(hashtext(id::text || ts::text || tf || COALESCE(period::text,''))::bigint) as checksum
   FROM returns_ema_multi_tf_u
   WHERE alignment_source = 'multi_tf'
   UNION ALL
   SELECT 'returns_bars_multi_tf_u', COUNT(*), SUM(hashtext(id::text || "timestamp"::text || tf)::bigint)
   FROM returns_bars_multi_tf_u
   WHERE alignment_source = 'multi_tf'
   UNION ALL
   SELECT 'returns_ama_multi_tf_u', COUNT(*), SUM(hashtext(id::text || ts::text || tf || indicator || params_hash)::bigint)
   FROM returns_ama_multi_tf_u
   WHERE alignment_source = 'multi_tf'
   ```
