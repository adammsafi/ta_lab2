Functions to implement:

1. `load_current_macro_regime(_engine) -> pd.DataFrame`:
   - Query: `SELECT * FROM cmc_macro_regimes WHERE date = (SELECT MAX(date) FROM cmc_macro_regimes WHERE profile = 'default') AND profile = 'default'`
   - Returns single row with: date, monetary_policy, liquidity, risk_appetite, carry, regime_key, macro_state

2. `load_macro_regime_history(_engine, days: int = 365) -> pd.DataFrame`:
   - Query: `SELECT date, monetary_policy, liquidity, risk_appetite, carry, regime_key, macro_state FROM cmc_macro_regimes WHERE profile = 'default' AND date >= CURRENT_DATE - :days ORDER BY date ASC`
   - Returns time series for timeline chart

3. `load_fred_freshness(_engine) -> pd.DataFrame`:
   - Query: `SELECT series_id, MAX(date) AS latest_date, COUNT(*) AS row_count FROM fred.series_values GROUP BY series_id ORDER BY series_id`
   - Add computed column `staleness_days` = (CURRENT_DATE - latest_date)
   - Add `frequency` column by mapping series_id to known frequency:
    - Daily: DFF, DGS2, DGS10, BAMLH0A0HYM2, VIXCLS, DTWEXBGS
    - Weekly: ICSA
    - Monthly: everything else (CPIAUCSL, UNRATE, M2SL, WALCL, etc.)
   - Add `fresh_threshold_days` based on frequency: daily=3, weekly=10, monthly=45
   - Add `status` column: "green" if staleness_days <= threshold, "orange" if staleness_days <= threshold*2.4, else "red"

4. `load_fred_series_quality(_engine) -> pd.DataFrame`:
   - Query: `SELECT series_id, date, value FROM fred.series_values ORDER BY series_id, date`
   - For each series: compute total rows, date range, gap count (missing expected dates based on frequency), coverage percentage
   - Return one row per series: series_id, first_date, last_date, total_rows, expected_rows, gap_count, coverage_pct

5. `load_macro_transition_log(_engine, days: int = 90) -> pd.DataFrame`:
   - Self-join to find transitions:
    ```sql
    SELECT curr.date, curr.regime_key, curr.macro_state,
           prev.regime_key AS prev_regime_key, prev.macro_state AS prev_macro_state,
           curr.monetary_policy, curr.liquidity, curr.risk_appetite, curr.carry,
           prev.monetary_policy AS prev_monetary_policy, prev.liquidity AS prev_liquidity,
           prev.risk_appetite AS prev_risk_appetite, prev.carry AS prev_carry
    FROM cmc_macro_regimes curr
    JOIN cmc_macro_regimes prev ON prev.date = curr.date - 1 AND prev.profile = curr.profile
    WHERE curr.profile = 'default'
      AND curr.date >= CURRENT_DATE - :days
      AND curr.regime_key != prev.regime_key
    ORDER BY curr.date DESC
    ```
