  <action>
  Add polars-native indicator functions to indicators.py. Keep ALL existing pandas functions UNCHANGED. Add new functions at the bottom in a clearly marked `# === Polars Variants ===` section.

  Import polars conditionally (same pattern as vol.py):
  ```python
  try:
      import polars as pl
      HAVE_POLARS = True
  except ImportError:
      pl = None
      HAVE_POLARS = False
  ```

  All polars functions operate on a single-group pl.DataFrame (pre-sorted by ts) and return pl.DataFrame with new columns:

  1. `rsi_polars(pl_df: pl.DataFrame, period: int = 14, price_col: str = "close", out_col: str | None = None) -> pl.DataFrame`:
     - delta = close.diff()
     - gain = delta.clip(lower_bound=0.0)
     - loss = (-delta).clip(lower_bound=0.0)
     - avg_gain = gain.ewm_mean(alpha=1/period, adjust=False, min_samples=1)
     - avg_loss = loss.ewm_mean(alpha=1/period, adjust=False, min_samples=1)
     - rs = avg_gain / avg_loss (replace 0 with null to avoid inf)
     - rsi = 100 - 100/(1+rs)
     - Research confirmed exact match with pandas.

  2. `macd_polars(pl_df: pl.DataFrame, fast=12, slow=26, signal=9, price_col="close", out_cols=None) -> pl.DataFrame`:
     - ema_fast = close.ewm_mean(span=fast, adjust=False)
     - ema_slow = close.ewm_mean(span=slow, adjust=False)
     - macd_line = ema_fast - ema_slow
     - signal_line = macd_line.ewm_mean(span=signal, adjust=False)
     - hist = macd_line - signal_line
     - Polars ewm_mean with span parameter: `half_life = None, alpha = 2/(span+1)` -- use alpha directly.

  3. `stoch_kd_polars(pl_df: pl.DataFrame, k=14, d=3, high_col="high", low_col="low", close_col="close", out_cols=None) -> pl.DataFrame`:
     - lowest = low.rolling_min(k, min_samples=k)
     - highest = high.rolling_max(k, min_samples=k)
     - k_line = 100 * (close - lowest) / (highest - lowest)
     - d_line = k_line.rolling_mean(d, min_samples=d)

  4. `bollinger_polars(pl_df: pl.DataFrame, window=20, price_col="close", n_sigma=2.0, out_cols=None) -> pl.DataFrame`:
     - ma = close.rolling_mean(window, min_samples=window)
     - std = close.rolling_std(window, min_samples=window)
     - upper = ma + n_sigma * std
     - lower = ma - n_sigma * std
     - bw = (upper - lower) / ma

  5. `atr_polars(pl_df: pl.DataFrame, period=14, high_col="high", low_col="low", close_col="close", out_col=None) -> pl.DataFrame`:
     - TR = max(h-lo, abs(h-prev_close), abs(lo-prev_close))
     - Uses rolling_mean (simple mean, matching the pandas indicators.py atr which uses rolling().mean(), NOT ewm)
     - NOTE: indicators.py atr() uses rolling mean, NOT ewm like vol.py add_atr(). Match the indicators.py version exactly.
     - `tr.rolling_mean(period, min_samples=period)`

  6. `adx_polars(pl_df: pl.DataFrame, period=14, high_col="high", low_col="low", close_col="close", out_col=None) -> pl.DataFrame`:
     - up = high.diff(), dn = -low.diff()
     - plus_dm = where((up > dn) & (up > 0), up, 0)
     - minus_dm = where((dn > up) & (dn > 0), dn, 0)
     - tr = true_range, atr_ = tr.rolling_mean(period)
     - plus_di = 100 * plus_dm.rolling_sum(period) / atr_
     - minus_di = 100 * minus_dm.rolling_sum(period) / atr_
     - dx = abs(plus_di - minus_di) / (plus_di + minus_di) * 100
     - adx = dx.rolling_mean(period)
     - Use pl.when/otherwise for conditional logic, not np.where

  CRITICAL NOTES from research:
  - ALL rolling calls use `min_samples=` (NOT `min_periods=`). Renamed in polars 1.21+.
  - polars ewm_mean: use `alpha=` parameter directly, NOT `span=` (polars doesn't have span parameter on ewm_mean).
    Conversion: alpha = 2 / (span + 1) for EMA.
  - For RSI/MACD: alpha = 1/period (Wilder smoothing) vs alpha = 2/(span+1) (EMA). Match the pandas version exactly.
  - polars diff() returns null on first row (pandas returns NaN). Both should produce identical downstream behavior since ewm_mean/rolling skip nulls.
