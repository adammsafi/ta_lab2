**1. _rolling_zscore() helper** (reusable for FRED-08 and FRED-12):
```python
def _rolling_zscore(series: pd.Series, window: int, min_fill_pct: float = 0.80) -> pd.Series:
    """Compute rolling z-score with minimum fill requirement (80% default)."""
    min_periods = max(1, int(min_fill_pct * window))
    roll_mean = series.rolling(window, min_periods=min_periods).mean()
    roll_std = series.rolling(window, min_periods=min_periods).std()
    return (series - roll_mean) / roll_std
```

**2. _compute_fed_regime() helper** (FRED-13 + FRED-16, modifies DataFrame in-place):
Implement exactly as specified in 66-RESEARCH.md Pattern 1:
- FRED-16: target_mid = (DFEDTARU + DFEDTARL) / 2, target_spread = DFEDTARU - DFEDTARL

- FRED-13 trajectory: DFF diff(90) > 0.25 = hiking, < -0.25 = cutting, else holding
- Graceful NaN handling: return None for NaN inputs
- If DFEDTARU/DFEDTARL missing, set target_mid/target_spread/fed_regime_structure to NaN/None
- If DFF missing, set fed_regime_trajectory to None

FRED-09 (Financial conditions):
- nfci_level = NFCI (direct copy)

- If NFCI not in columns, set both to NaN/None

FRED-10 (M2 money supply):
- m2_yoy_pct = M2SL.pct_change(365) * 100.0
- CRITICAL: Use pct_change(365) NOT pct_change(1) -- M2SL is monthly forward-filled to daily, pct_change(1) would give 0 on non-release days
- If M2SL not in columns, set to NaN

FRED-11 (Carry trade):
- dexjpus_level = DEXJPUS (direct copy)
- dexjpus_5d_pct_change = DEXJPUS.pct_change(5) * 100.0
- dexjpus_20d_vol = DEXJPUS.pct_change(1).rolling(20, min_periods=16).std() * 100.0
- dexjpus_daily_zscore = z-score of DEXJPUS 1-day return using 20d rolling window (min_periods=16)
  Compute: daily_move = DEXJPUS.pct_change(1) * 100.0, then use rolling(20, min_periods=16) mean and std
- If DEXJPUS not in columns, set all four to NaN

FRED-12 (Net liquidity z-score + trend):
- net_liquidity_365d_zscore = _rolling_zscore(net_liquidity, 365) -- requires net_liquidity column from Phase 65
- net_liquidity_trend = "expanding" if 30d MA > 150d MA, "contracting" if 30d MA < 150d MA, None if NaN
  Use ma30 = net_liquidity.rolling(30, min_periods=24).mean(), ma150 = net_liquidity.rolling(150, min_periods=120).mean()
- If net_liquidity not in columns, set both to NaN/None

FRED-13/16 (Fed regime):
- Call _compute_fed_regime(result) which modifies in-place, adding:
  target_mid, target_spread, fed_regime_structure, fed_regime_trajectory

FRED-14 (Carry momentum):
- Uses dexjpus_daily_zscore (computed in FRED-11 above) and us_jp_rate_spread (from Phase 65)
- If carry_spread (us_jp_rate_spread) exists and is positive, threshold = 2.0; otherwise threshold = 1.5
- carry_momentum = 1.0 where abs(dexjpus_daily_zscore) > threshold, else 0.0
- Set carry_momentum to None where dexjpus_daily_zscore is NaN
- If dexjpus_daily_zscore not computed, set carry_momentum to NaN

FRED-15 (CPI surprise proxy):
- cpi_mom = CPIAUCSL.pct_change(30) * 100.0 (approx MoM from monthly ffilled data)
- baseline = cpi_mom.rolling(90, min_periods=72).mean() (3-month trend)
- cpi_surprise_proxy = cpi_mom - baseline
- If CPIAUCSL not in columns, set to NaN
