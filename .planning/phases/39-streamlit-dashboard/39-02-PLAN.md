4. `build_regime_price_chart(close_series: pd.Series, regimes_df: pd.DataFrame) -> go.Figure`:
   - close_series: indexed by tz-aware UTC timestamps, values are close prices
   - regimes_df: has columns ts, trend_state (derived from l2_label via split_part in the query layer)
   - Create a go.Figure with a Scatter trace for the close price line (white line, width 1.5)
   - Add colored background bands using `fig.add_vrect()` for each regime period:
     - Iterate over regimes_df rows. For each row, the band starts at row["ts"] and ends at the NEXT row's ts (or close_series.index[-1] for the last row)
     - Color mapping (REGIME_COLORS dict at module level):
       ```python
       REGIME_COLORS = {
           "Up": "rgba(0, 200, 100, 0.15)",       # green, low opacity
           "Down": "rgba(220, 50, 50, 0.15)",      # red, low opacity
           "Sideways": "rgba(150, 150, 150, 0.12)", # gray, low opacity
       }
       ```
     - For each vrect: `fig.add_vrect(x0=start_ts, x1=end_ts, fillcolor=color, opacity=1, layer="below", line_width=0)`
     - Use `REGIME_COLORS.get(trend_state, REGIME_COLORS["Sideways"])` for unknown trends
   - Apply `fig.update_layout(template="plotly_dark", title="Price with Regime Overlay", xaxis_title="Date", yaxis_title="Close Price")`
   - Return figure

5. `build_regime_timeline(regimes_df: pd.DataFrame) -> go.Figure`:
   - regimes_df: has columns ts, trend_state, vol_state (from query layer)
   - Build a horizontal bar chart showing regime periods over time
   - One bar per regime period, colored by trend_state using REGIME_COLORS (but higher opacity for standalone chart):
     ```python
     REGIME_BAR_COLORS = {
         "Up": "rgb(0, 200, 100)",
         "Down": "rgb(220, 50, 50)",
         "Sideways": "rgb(150, 150, 150)",
     }
     ```
   - Each bar spans from the regime start_ts to the next regime start_ts (or last bar for final period)
   - Use go.Bar with orientation="h", x=[duration_days], base=[start_ts], color per bar
   - Actually, simpler approach: use go.Figure with add_trace for each regime segment:
     - Convert consecutive same-regime rows into regime blocks (aggregate consecutive rows with same trend_state)
     - For each block: add a horizontal bar from block_start to block_end
   - Apply `fig.update_layout(template="plotly_dark", title="Regime Timeline", xaxis_title="Date", yaxis_title="", showlegend=True)`
   - Alternative simpler approach if horizontal bars are complex: Use a colored scatter strip chart with markers at each timestamp colored by regime. Or use Plotly timeline (px.timeline) if available.
   - RECOMMENDED: Use a simple approach -- for each row in regimes_df, plot a colored marker. Use go.Scatter with mode="markers" where marker color maps from trend_state. This is simpler and still effective:
     ```python
     fig = go.Figure()
     for trend, color in REGIME_BAR_COLORS.items():
         mask = regimes_df["trend_state"] == trend
         subset = regimes_df[mask]
         fig.add_trace(go.Scatter(
             x=subset["ts"], y=[trend] * len(subset),
             mode="markers", name=trend,
             marker=dict(color=color, size=8, symbol="square"),
         ))
     ```
   - Apply plotly_dark template
   - Return figure

Handle edge cases:
