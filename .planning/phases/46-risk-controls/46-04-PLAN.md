   - _is_circuit_breaker_tripped returns True
   - _load_limits returns low caps

   Call check_order. Verify that the FIRST reason returned is "kill_switch_active" (kill switch takes priority over everything).

   Then change _is_halted to return False.

   Then change _is_circuit_breaker_tripped to return False.
