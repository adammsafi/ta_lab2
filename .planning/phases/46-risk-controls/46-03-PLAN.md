**OverrideInfo dataclass:**
- override_id: str
- asset_id: int

- reason: str

- sticky: bool
- created_at: datetime
- applied_at: Optional[datetime] = None
- reverted_at: Optional[datetime] = None
- revert_reason: Optional[str] = None

**OverrideManager class:**
Constructor takes `engine: Engine`. Store as `self._engine`. Create logger.

`create_override(self, asset_id: int, strategy_id: int, operator: str, reason: str, system_signal: str, override_action: str, sticky: bool = False) -> str`:
Within engine.begin():
1. INSERT into cmc_risk_overrides (asset_id, strategy_id, operator, reason, system_signal, override_action, sticky) RETURNING override_id
2. INSERT into cmc_risk_events (event_type='override_created', trigger_source='manual', reason, operator, asset_id, strategy_id, override_id)
3. Return override_id as string
Log the creation with INFO level.

`apply_override(self, override_id: str) -> None`:
Within engine.begin():
1. UPDATE cmc_risk_overrides SET applied_at = now() WHERE override_id = :id AND applied_at IS NULL
2. If no row updated (already applied or not found), log warning and return
3. INSERT into cmc_risk_events (event_type='override_applied', trigger_source='system', reason='Override applied by executor', override_id)

`revert_override(self, override_id: str, reason: str, operator: str) -> None`:
Within engine.begin():
1. UPDATE cmc_risk_overrides SET reverted_at = now(), revert_reason = :reason WHERE override_id = :id AND reverted_at IS NULL
2. If no row updated (already reverted or not found), log warning and return
3. INSERT into cmc_risk_events (event_type='override_reverted', trigger_source='manual', reason, operator, override_id)

`get_active_overrides(self, asset_id: Optional[int] = None, strategy_id: Optional[int] = None) -> list[OverrideInfo]`:
SELECT from cmc_risk_overrides WHERE reverted_at IS NULL.
- If asset_id provided: AND asset_id = :asset_id
- If strategy_id provided: AND strategy_id = :strategy_id
- ORDER BY created_at DESC
Return list of OverrideInfo dataclass instances.

`get_pending_non_sticky_overrides(self) -> list[OverrideInfo]`:
SELECT from cmc_risk_overrides WHERE reverted_at IS NULL AND sticky = FALSE AND applied_at IS NOT NULL.
These are the overrides the executor should auto-revert after the signal cycle.

`get_override(self, override_id: str) -> Optional[OverrideInfo]`:
SELECT single override by override_id. Return None if not found.
