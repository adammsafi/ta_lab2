    For every subprocess.run() call in src/ta_lab2/scripts/, add a timeout= parameter using tiered values based on operation type:

    Timeout tiers (define as module-level constants where appropriate):
    - TIMEOUT_BARS = 7200       (2 hours -- bar builders can be slow for full rebuilds)
    - TIMEOUT_EMAS = 3600       (1 hour -- EMA refreshers)
    - TIMEOUT_REGIMES = 1800    (30 minutes -- regime refresher)
    - TIMEOUT_STATS = 3600      (1 hour -- stats runners)
    - TIMEOUT_AUDIT = 1800      (30 minutes -- audit scripts)
    - TIMEOUT_SYNC = 600        (10 minutes -- sync/setup scripts)
    - TIMEOUT_GIT = 30          (30 seconds -- git commands)
    - TIMEOUT_BASELINE_BARS = 7200  (2 hours -- baseline bar builders)
    - TIMEOUT_BASELINE_EMAS = 3600  (1 hour -- baseline EMA refreshers)

    For each file, the pattern is:
    1. Add `import subprocess` if not already present (it always is)
    2. Add timeout= to every subprocess.run() call
    3. Wrap in try/except subprocess.TimeoutExpired if not already wrapped

    The existing code pattern is ALWAYS one of:
    ```python
    # verbose mode
    result = subprocess.run(cmd, check=False)
    # capture mode
    result = subprocess.run(cmd, check=False, capture_output=True, text=True)
    ```

    Transform to:
    ```python
    try:
        if verbose:
            result = subprocess.run(cmd, check=False, timeout=TIMEOUT_X)
        else:
            result = subprocess.run(cmd, check=False, capture_output=True, text=True, timeout=TIMEOUT_X)
        # ... existing success/failure handling unchanged ...
    except subprocess.TimeoutExpired:
        duration = time.perf_counter() - start
        error_msg = f"Timed out after {TIMEOUT_X}s"
        print(f"\n[TIMEOUT] {component_name}: {error_msg}")
        return ComponentResult(
            component=component_name,
            success=False,
            duration_sec=duration,
            returncode=-1,
            error_message=error_msg,
        )
    ```

    IMPORTANT: Many files already have try/except Exception wrapping. Add TimeoutExpired as a SEPARATE except clause BEFORE the generic Exception handler. Do NOT nest a new try/except inside the existing one -- add TimeoutExpired to the existing try block.
