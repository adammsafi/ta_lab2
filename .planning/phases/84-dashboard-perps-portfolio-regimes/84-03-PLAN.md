Create `src/ta_lab2/dashboard/queries/ama.py` following the exact pattern from `queries/research.py`:
- `from __future__ import annotations`, `import pandas as pd`, `import streamlit as st`, `from sqlalchemy import text`
- All functions: `@st.cache_data(ttl=...)`, `_engine` prefix

Implement these 3 functions:

1. `load_ama_params_catalogue(_engine) -> pd.DataFrame` -- all AMA indicator/param combos for UI selector.
   TTL=3600 (dimension data, rarely changes). 18 rows total.
   ```sql
   SELECT indicator, params_hash, label, params_json
   FROM public.dim_ama_params
   ORDER BY indicator, label
   ```

2. `load_ama_curves(_engine, asset_id: int, tf: str = "1D", indicator: str = "KAMA", days_back: int = 365) -> pd.DataFrame` -- AMA curves with human-readable labels from dim_ama_params.
   CRITICAL: er column is NULL for DEMA/HMA/TEMA -- only KAMA computes efficiency ratio.
   Always filter by indicator AND alignment_source='multi_tf' AND roll=false.
   ```sql
   SELECT a.ts, p.label, a.indicator, a.ama, a.d1, a.d2,
          a.d1_roll, a.d2_roll, a.er, a.roll
   FROM public.ama_multi_tf_u a
   JOIN public.dim_ama_params p
     ON p.params_hash = a.params_hash AND p.indicator = a.indicator
   WHERE a.id = :id
     AND a.tf = :tf
     AND a.indicator = :indicator
     AND a.alignment_source = 'multi_tf'
     AND a.roll = false
     AND a.ts >= NOW() - (:days_back || ' days')::interval
   ORDER BY a.ts, p.label
   ```
   Convert ts: `pd.to_datetime(df["ts"], utc=True)`

3. `load_ema_for_comparison(_engine, asset_id: int, tf: str = "1D", periods: list[int] | None = None, days_back: int = 365) -> pd.DataFrame` -- fixed EMA values for AMA vs EMA comparison.
   alignment_source='multi_tf' is canonical. EMA column is `ema` (NOT `ema_value` in the table).
   ```sql
   SELECT ts, period, ema
   FROM public.ema_multi_tf_u
   WHERE id = :id
     AND tf = :tf
     AND alignment_source = 'multi_tf'
     AND period = ANY(:periods)
     AND ts >= NOW() - (:days_back || ' days')::interval
   ORDER BY period, ts
   ```
   Default periods: [9, 21, 50, 200]. Convert ts: `pd.to_datetime(df["ts"], utc=True)`

CRITICAL gotchas:
- ama_multi_tf_u is 170M rows -- always filter by indicator, asset, tf, and days_back
- er is NULL for DEMA, HMA, TEMA -- only KAMA computes it
- ema_multi_tf_u has NO d1/d2 columns -- only ama_multi_tf_u has d1, d2, d1_roll, d2_roll
- dim_ama_params has 18 rows: DEMA x5 + HMA x5 + KAMA x3 + TEMA x5
