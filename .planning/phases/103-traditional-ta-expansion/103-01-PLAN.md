**10 indicator functions (Batch 1) following the exact API convention from indicators.py:**

1. **ichimoku(obj, *, tenkan=9, kijun=26, senkou_b=52, high_col, low_col, close_col, out_cols, inplace)** -> DataFrame
   - 5 outputs: ichimoku_tenkan, ichimoku_kijun, ichimoku_span_a, ichimoku_span_b, ichimoku_chikou
   - tenkan = (rolling_max(high,9) + rolling_min(low,9)) / 2
   - kijun = (rolling_max(high,26) + rolling_min(low,26)) / 2
   - span_a = (tenkan + kijun) / 2 -- NO forward shift
   - span_b = (rolling_max(high,52) + rolling_min(low,52)) / 2 -- NO forward shift
   - chikou = close (current close, no shift -- for IC testing the "chikou" is just close at T, which is redundant; include for completeness but shift(26) backward to get close at T-26 is valid as feature)
   - Actually per research: chikou = close.shift(26) gives close at T-26, which is historical info and valid. Use `close.shift(26)`.

2. **williams_r(obj, window=14, *, high_col, low_col, close_col, out_col, inplace)** -> Series
   - `willr_N = -100 * (HH_N - close) / (HH_N - LL_N)`

3. **keltner(obj, *, ema_period=20, atr_period=10, multiplier=2.0, high_col, low_col, close_col, out_cols, inplace)** -> DataFrame
   - 4 outputs: kc_mid_20, kc_upper_20, kc_lower_20, kc_width_20
   - mid = _ema(close, 20), upper = mid + 2*ATR(10), lower = mid - 2*ATR(10), width = (upper-lower)/mid

4. **cci(obj, window=20, *, high_col, low_col, close_col, out_col, inplace)** -> Series
   - `tp = _tp(H,L,C); sma_tp = _sma(tp,20); mean_dev = (tp-sma_tp).abs().rolling(20).mean(); cci = (tp-sma_tp)/(0.015*mean_dev)`
   - CRITICAL: Use mean absolute deviation, NOT rolling std

5. **elder_ray(obj, *, period=13, high_col, low_col, close_col, out_cols, inplace)** -> DataFrame
   - 2 outputs: elder_bull_13, elder_bear_13
   - bull = high - _ema(close,13), bear = low - _ema(close,13)

6. **force_index(obj, *, smooth=13, close_col, volume_col, out_cols, inplace)** -> DataFrame
   - 2 outputs: fi_1, fi_13
   - fi_1 = close.diff() * volume; fi_13 = _ema(fi_1, 13)

7. **vwap(obj, window=14, *, high_col, low_col, close_col, volume_col, out_cols, inplace)** -> DataFrame
   - 2 outputs: vwap_14, vwap_dev_14
   - vwap_N = rolling_sum(tp*vol, N) / rolling_sum(vol, N)
   - vwap_dev_N = close / vwap_N - 1
   - CRITICAL: Rolling window, NOT cumulative

8. **cmf(obj, window=20, *, high_col, low_col, close_col, volume_col, out_col, inplace)** -> Series
   - mfm = ((close-low)-(high-close))/(high-low); mfv = mfm*volume
   - cmf_20 = rolling_sum(mfv,20) / rolling_sum(volume,20)

9. **chaikin_osc(obj, *, fast=3, slow=10, high_col, low_col, close_col, volume_col, out_col, inplace)** -> Series
   - mfm = ((close-low)-(high-close))/(high-low); mfv = mfm*volume; adl = cumsum(mfv)
   - chaikin_osc = _ema(adl, 3) - _ema(adl, 10)

10. **hurst(obj, window=100, *, max_lag=20, close_col, out_col, inplace)** -> Series
    - Variance-scaling method: for lags 2..max_lag, compute std of lagged differences, fit log-log polyfit slope
    - `rolling(100, min_periods=100).apply(_hurst_inner, raw=True)`
    - Output: hurst_100

11. **vidya(obj, *, cmo_period=9, vidya_period=9, close_col, out_col, inplace)** -> Series
    - MUST use explicit Python loop (not vectorizable)
    - CMO = (UpSum - DnSum) / (UpSum + DnSum) over rolling cmo_period
    - k = 2/(vidya_period+1); alpha_i = |CMO_i| * k
    - result[i] = alpha * close[i] + (1-alpha) * result[i-1]
    - Initialize at first valid CMO bar with close value

12. **frama(obj, *, period=16, close_col, out_col, inplace)** -> Series
    - MUST use explicit Python loop (not vectorizable)
    - period must be even (assert or force)
    - Fractal dimension D from half-window ranges: N1, N2, N3
    - alpha = exp(-4.6 * (D-1)), clipped to [0.01, 1.0]
    - result[i] = alpha * close[i] + (1-alpha) * result[i-1]

13. **aroon(obj, window=25, *, high_col, low_col, out_cols, inplace)** -> DataFrame
    - 3 outputs: aroon_up_25, aroon_dn_25, aroon_osc_25
    - Rolling window = N+1 (26 for Aroon-25)
    - aroon_up = high.rolling(N+1).apply(lambda x: (N - x[:-1].argmax()) / N * 100)
    - aroon_dn = low.rolling(N+1).apply(lambda x: (N - x[:-1].argmin()) / N * 100)
    - aroon_osc = aroon_up - aroon_dn
    - CRITICAL: Use N+1 rolling window, per Pitfall 6 in research

14. **trix(obj, *, period=15, signal_period=9, close_col, out_cols, inplace)** -> DataFrame
    - 2 outputs: trix_15, trix_signal_9
    - ema1 = _ema(close, N); ema2 = _ema(ema1, N); ema3 = _ema(ema2, N)
    - trix = (ema3 - ema3.shift(1)) / ema3.shift(1) * 100
    - trix_signal = _ema(trix, signal_period)

15. **ultimate_osc(obj, *, p1=7, p2=14, p3=28, high_col, low_col, close_col, out_col, inplace)** -> Series
    - prev_close = close.shift(1)
    - bp = close - min(low, prev_close)
    - tr = max(high, prev_close) - min(low, prev_close)
    - avg_i = bp.rolling(pi).sum() / tr.rolling(pi).sum()
    - uo = 100 * (4*avg7 + 2*avg14 + avg28) / 7

16. **vortex(obj, window=14, *, high_col, low_col, close_col, out_cols, inplace)** -> DataFrame
    - 2 outputs: vi_plus_14, vi_minus_14
    - vm_plus = |high - low.shift(1)|; vm_minus = |low - high.shift(1)|
    - tr_sum = _tr(H,L,C).rolling(14).sum()
    - vi_plus = vm_plus.rolling(14).sum() / tr_sum; vi_minus = vm_minus.rolling(14).sum() / tr_sum

17. **emv(obj, window=14, *, high_col, low_col, volume_col, out_cols, inplace)** -> DataFrame
    - 2 outputs: emv_1, emv_14
    - midpoint_move = ((H+L)/2) - ((H.shift(1)+L.shift(1))/2)
    - box_ratio = (volume/1e8) / (H-L).replace(0, np.nan)
    - emv_1 = midpoint_move / box_ratio
    - emv_14 = _sma(emv_1, 14)

18. **mass_index(obj, *, ema_period=9, sum_period=25, high_col, low_col, out_col, inplace)** -> Series
    - hl = high - low; ema1 = _ema(hl, 9); ema2 = _ema(ema1, 9)
    - ratio = ema1 / ema2.replace(0, np.nan)
    - mass_idx = ratio.rolling(25).sum()

19. **kst(obj, *, close_col, out_cols, inplace)** -> DataFrame
    - 2 outputs: kst, kst_signal
    - ROC periods: 10, 15, 20, 30; SMA periods: 10, 10, 10, 15; weights: 1, 2, 3, 4
    - roc(close, n) = (close / close.shift(n) - 1) * 100
    - kst = 1*SMA(ROC10,10) + 2*SMA(ROC15,10) + 3*SMA(ROC20,10) + 4*SMA(ROC30,15)
    - kst_signal = _sma(kst, 9)

20. **coppock(obj, *, roc_long=14, roc_short=11, wma_period=10, close_col, out_col, inplace)** -> Series
    - roc14 = (close/close.shift(14) - 1) * 100
    - roc11 = (close/close.shift(11) - 1) * 100
    - coppock = _wma(roc14 + roc11, 10)
