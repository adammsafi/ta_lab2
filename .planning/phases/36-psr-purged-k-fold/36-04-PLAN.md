<task type="auto">
  <name>Task 2: Create standalone compute_psr.py CLI</name>
  <files>src/ta_lab2/scripts/backtests/compute_psr.py</files>
  <action>
Create `src/ta_lab2/scripts/backtests/compute_psr.py` as a standalone CLI script for computing/recomputing PSR on historical backtest runs.

The script must:

1. Accept CLI args via argparse:
   - `--run-id UUID` : compute PSR for a single run
   - `--all` : compute PSR for all runs that do NOT have a psr_results row yet
   - `--recompute` : force recompute even if psr_results row exists (use with --all)
   - `--sr-star FLOAT` : benchmark Sharpe (default 0.0, annualized -- converted internally to per-bar)
   - `--dry-run` : print what would be computed, do not write to DB

2. For each run_id:
   a. Load returns from the backtest: query cmc_backtest_trades to reconstruct per-bar returns, OR load from the signal-based price data. The simplest approach: load the run metadata from cmc_backtest_runs (asset_id, start_ts, end_ts), load prices from cmc_features (same as backtest_from_signals.load_prices), compute equity curve, extract daily returns.

   b. Alternatively (simpler): query cmc_backtest_trades for the run, use entry_ts/exit_ts/pnl_pct to reconstruct the returns series. If no trades exist, skip.

   Use approach (a) via rebuilding the price-based returns:
   - Load run metadata: `SELECT asset_id, start_ts, end_ts FROM cmc_backtest_runs WHERE run_id = :run_id`
   - Load prices: `SELECT ts, close FROM cmc_features WHERE id = :asset_id AND tf = '1D' AND ts BETWEEN :start_ts AND :end_ts ORDER BY ts`
   - Compute returns: `prices['close'].pct_change().dropna()`
   - Compute PSR: `compute_psr(returns.values, sr_star=sr_star_per_bar)`
   - Compute MinTRL: `min_trl(returns.values, sr_star=sr_star_per_bar, target_psr=0.95, freq_per_year=365)`
   - Write to psr_results (ON CONFLICT DO UPDATE on run_id + formula_version)
   - Update cmc_backtest_metrics.psr column

3. Use NullPool for DB connection (matching project pattern).
4. Use resolve_db_url() from refresh_utils.
5. Log summary at end: "Computed PSR for N runs, skipped M (already computed), failed K"

Invocation pattern: `python -m ta_lab2.scripts.backtests.compute_psr --run-id <UUID>` or `python -m ta_lab2.scripts.backtests.compute_psr --all`

Handle the sr_star conversion: if user passes --sr-star 1.0 (annual), convert to per-bar: `sr_star_per_bar = sr_star / np.sqrt(365)` (for daily backtests with freq_per_year=365).
  </action>
