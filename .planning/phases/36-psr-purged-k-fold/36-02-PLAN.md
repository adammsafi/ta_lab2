    - compute_dsr(best_returns, sr_estimates=[list of N SR values]) -> float <= compute_psr(best_returns, sr_star=0)

    - compute_dsr(best_returns, sr_estimates=None, n_trials=None) -> raises ValueError

    - min_trl(strong_positive_returns, sr_star=0) -> dict with n_obs > 0, calendar_days > 0
    - min_trl(returns_where_sr_leq_sr_star) -> dict with n_obs=inf, calendar_days=inf
    - min_trl(returns, target_psr=0.99) -> higher n_obs than min_trl(returns, target_psr=0.95)

  </behavior>
  <implementation>
    Create `src/ta_lab2/backtests/psr.py` with:

    1. `compute_psr(returns, sr_star=0.0) -> float`:
       - Guard: n < 30 -> NaN + warnings.warn; n < 100 -> warnings.warn
       - sr_hat = mean(returns) / std(returns, ddof=1)
       - gamma_3 = scipy.stats.skew(returns)
       - gamma_4 = scipy.stats.kurtosis(returns, fisher=False)  # PEARSON kurtosis CRITICAL
       - var_sr = (1 - gamma_3 * sr_hat + (gamma_4 - 1)/4 * sr_hat^2) / (n - 1)
       - Guard: var_sr <= 0 -> NaN
       - z = (sr_hat - sr_star) / sqrt(var_sr)
       - return norm.cdf(z)

    2. `expected_max_sr(sr_estimates, expected_mean=0.0) -> float`:
       - N = len(sr_estimates)
       - euler_gamma = 0.5772156649
       - std_sr = np.std(sr_estimates - expected_mean, ddof=1)
       - e_max = expected_mean + std_sr * ((1-gamma)*norm.ppf(1-1/N) + gamma*norm.ppf(1-1/(N*e)))

    3. `compute_dsr(best_trial_returns, sr_estimates=None, n_trials=None, sr_star_override=None) -> float`:
       - Exact mode: benchmark = expected_max_sr(sr_estimates)
       - Approx mode: benchmark via Bailey formula with assumed std=1
       - Call compute_psr(best_trial_returns, sr_star=benchmark)

    4. `min_trl(returns, sr_star=0.0, target_psr=0.95, freq_per_year=365) -> dict`:
       - Guard: sr_hat <= sr_star -> return inf values
       - v_factor = 1 - gamma_3*sr_hat + (gamma_4-1)/4*sr_hat^2
       - z_threshold = norm.ppf(target_psr)
       - n_obs = (z_threshold * sqrt(v_factor) / (sr_hat - sr_star))^2 + 1
       - calendar_days = round(ceil(n_obs) / freq_per_year * 365)

    Follow TDD: write failing tests first (RED), implement to pass (GREEN), refactor if needed.
  </implementation>
