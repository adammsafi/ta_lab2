Signature:
```python
def load_ctf_features(
    conn,
    asset_id: int,
    base_tf: str,
    train_start: pd.Timestamp,
    train_end: pd.Timestamp,
    *,
    ref_tfs: Optional[list[str]] = None,
    indicator_names: Optional[list[str]] = None,
    alignment_source: str = "multi_tf",
    venue_id: int = 1,
) -> pd.DataFrame:
```

Implementation details:
1. SQL query: SELECT c.ts, d.indicator_name, c.ref_tf, c.ref_value, c.base_value, c.slope, c.divergence, c.agreement, c.crossover FROM public.ctf c JOIN public.dim_ctf_indicators d ON d.indicator_id = c.indicator_id WHERE c.id = :asset_id AND c.base_tf = :base_tf AND c.ts >= :train_start AND c.ts <= :train_end AND c.alignment_source = :alignment_source AND c.venue_id = :venue_id ORDER BY c.ts
2. If ref_tfs is provided, add AND c.ref_tf = ANY(:ref_tfs) to the WHERE clause (use text() with bindparam or build the query conditionally)
3. If indicator_names is provided, add AND d.indicator_name = ANY(:indicator_names)
4. After loading: apply pd.to_datetime(df["ts"], utc=True) to fix Windows tz-naive bug
5. Return empty DataFrame if no rows
6. Use VECTORIZED pivot approach (not row-by-row iterrows):
   - df["ref_tf_lower"] = df["ref_tf"].str.lower()
   - df["col_base"] = df["indicator_name"] + "_" + df["ref_tf_lower"]
   - composite_cols = ["ref_value", "base_value", "slope", "divergence", "agreement", "crossover"]
   - melted = df.melt(id_vars=["ts", "col_base"], value_vars=composite_cols, var_name="composite", value_name="val")
   - melted["feature_col"] = melted["col_base"] + "_" + melted["composite"]
   - wide = melted.pivot_table(index="ts", columns="feature_col", values="val", aggfunc="first")
   - wide.columns.name = None
   - wide.index.name = "ts"
7. Drop all-NaN columns: wide = wide.dropna(axis=1, how="all")
8. Return wide DataFrame

IMPORTANT: Do NOT include a `close` column in the output -- batch_compute_ic() takes close as a separate Series.
IMPORTANT: The output must have a DatetimeIndex that is tz-aware UTC.
IMPORTANT: Use `from sqlalchemy import text` for the SQL (already imported in this file).
