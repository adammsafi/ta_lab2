    After the sweep completes, verify IC results coverage:
    ```python
    from ta_lab2.io import get_engine
    from sqlalchemy import text
    engine = get_engine()
    with engine.connect() as conn:
        # Count distinct assets in CTF IC results
        r = conn.execute(text('''
            SELECT COUNT(DISTINCT asset_id) as n_assets,
                   COUNT(*) as n_rows
            FROM public.ic_results
            WHERE feature LIKE '%%\_7d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_14d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_30d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_90d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_180d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_365d\_%%' ESCAPE '\\'
        ''')).fetchone()
        print(f'CTF IC results: {r[1]} rows covering {r[0]} distinct assets')

        # Breakdown by base_tf
        tfs = conn.execute(text('''
            SELECT tf, COUNT(DISTINCT asset_id) as n_assets, COUNT(*) as n_rows
            FROM public.ic_results
            WHERE feature LIKE '%%\_7d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_14d\_%%' ESCAPE '\\'
               OR feature LIKE '%%\_30d\_%%' ESCAPE '\\'
            GROUP BY tf
            ORDER BY tf
        ''')).fetchall()
        for t in tfs:
            print(f'  tf={t[0]}: {t[1]} assets, {t[2]} rows')
    ```
  </action>
  <verify>

    2. IC results cover >= 10 distinct asset_ids for CTF features (gap 1 closure).
