<task type="auto">
  <name>Task 1: Create portfolio query module</name>
  <files>src/ta_lab2/dashboard/queries/portfolio.py</files>
  <action>
Create `src/ta_lab2/dashboard/queries/portfolio.py` with three cached query functions
following the exact project pattern (see `queries/trading.py` and `queries/research.py`
for reference):

1. `load_available_optimizers(_engine) -> list[str]`
   - `@st.cache_data(ttl=300)`
   - SQL: `SELECT DISTINCT optimizer FROM public.portfolio_allocations ORDER BY optimizer`
   - Returns list of optimizer name strings
   - Returns empty list if table is empty

2. `load_latest_allocations(_engine, optimizer: str = "hrp") -> pd.DataFrame`
   - `@st.cache_data(ttl=300)`
   - SQL: DISTINCT ON (asset_id) from portfolio_allocations WHERE optimizer = :optimizer,
     ordered by asset_id, ts DESC (most recent row per asset)
   - JOIN public.dim_assets a ON a.id = pa.asset_id for symbol lookup
   - CAST weight and final_weight to FLOAT (they are NUMERIC in the schema)
   - Columns returned: symbol, asset_id, ts, optimizer, weight, final_weight,
     regime_label, condition_number
   - Convert ts to pd.to_datetime(utc=True) after load
   - Return empty DataFrame if no rows

3. `load_allocation_history(_engine, optimizer: str = "hrp", days: int = 30) -> pd.DataFrame`
   - `@st.cache_data(ttl=300)`
   - SQL: SELECT symbol, ts, CAST(weight AS FLOAT) from portfolio_allocations
     JOIN dim_assets, WHERE optimizer = :optimizer AND ts >= NOW() - (:days * INTERVAL '1 day')
   - Pivot result: index=ts (DatetimeIndex), columns=symbol, values=weight
   - Return empty DataFrame if no rows
   - Convert ts to pd.to_datetime(utc=True) before pivot

All functions: use `_engine` (underscore prefix) as first arg, use `text()` from sqlalchemy,
use `with _engine.connect() as conn:` pattern, docstrings explaining columns returned.

Module docstring: "Cached query functions for Portfolio Allocation page."
Imports: `from __future__ import annotations`, pandas, streamlit, sqlalchemy.text
  </action>
