3. Create 4-column traffic light indicators:

   a. **Kill Switch** (column 1):
      - Load risk_state
      - If trading_state == 'active': st.metric("Kill Switch", "OK", delta="Active", delta_color="normal")
      - If trading_state == 'halted': st.metric("Kill Switch", "HALTED", delta="Since " + halted_at, delta_color="inverse")

   b. **Drift Pause** (column 2):
      - If not drift_paused: st.metric("Drift Pause", "OK", delta="Monitoring", delta_color="normal")
      - If drift_paused:
        - Compute days_paused = (now - drift_paused_at).days
        - If days_paused < 3: st.metric("Drift Pause", "PAUSED", delta=f"{days_paused}d", delta_color="off")
        - Else: st.metric("Drift Pause", "PAUSED", delta=f"{days_paused}d - Escalate!", delta_color="inverse")

   c. **Executor Last Run** (column 3):
      - Load executor_run_log(engine, limit=1)
      - If empty or no runs: st.metric("Executor", "NO DATA", delta_color="off")
      - Else: compute hours_since = (now - finished_at).total_seconds() / 3600
        - If < 26: st.metric("Executor", "OK", delta=f"{hours_since:.0f}h ago", delta_color="normal")
        - If 26-48: st.metric("Executor", "STALE", delta=f"{hours_since:.0f}h ago", delta_color="off")
        - If > 48: st.metric("Executor", "STALE", delta=f"{hours_since:.0f}h ago", delta_color="inverse")

   d. **Circuit Breaker** (column 4):
      - Parse cb_breaker_tripped_at from risk_state (json.loads if not None/empty)
      - If no keys or empty: st.metric("Circuit Breaker", "OK", delta="0 tripped", delta_color="normal")
      - If any keys tripped: st.metric("Circuit Breaker", "TRIPPED", delta=f"{n} keys", delta_color="inverse")
