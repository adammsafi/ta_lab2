```python
AUTO_REFRESH_SECONDS = 900
```

**Page layout (top to bottom):**

1. **Alert banners** (outside fragment):
   - Load risk_state via load_risk_state(engine)
   - If drift_paused: st.warning() with drift_paused_at and drift_paused_reason
   - If trading_state == 'halted': st.error() with halted info

2. **Sidebar controls** (outside fragment):
   - Load active executor configs via load_executor_configs(engine)
   - st.sidebar.selectbox "Strategy" -- populated from config_name values (or "All" if multiple)
   - st.sidebar.slider "History (days)" -- 7, 14, 30, 60, 90 (default 30)
   - st.sidebar.checkbox "Show 7-day history table" (default False)

3. **Auto-refreshing content section** wrapped in `@st.fragment(run_every=AUTO_REFRESH_SECONDS)`:
   Fragment receives `_engine`, `selected_config_id`, `history_days`, `show_history`.

   a. **Drift Summary Cards** (st.columns(4)):
      - Load drift_summary
      - If selected config: filter to that config_id
      - Current TE 5d: current_tracking_error_5d with delta from avg
      - Current TE 30d: latest from timeseries (or from summary avg_tracking_error_30d)
      - Days Monitored: days_monitored from summary
      - Breaches: breach_count from summary

   b. **Tracking Error Chart** (DASH-L04):
      - Load drift_timeseries for selected config and history_days
      - Load risk_limits to get threshold_5d and threshold_30d values
      - Call build_tracking_error_chart(drift_df, threshold_5d, threshold_30d)
      - st.plotly_chart(fig, use_container_width=True, theme=None)
      - chart_download_button for export
      - st.caption(f"Data as of {last_metric_date}") to show materialized view freshness

   c. **Equity Overlay Chart**:
      - Call build_equity_overlay_chart(drift_df) -- uses same drift timeseries data
      - st.plotly_chart(fig, use_container_width=True, theme=None)
      - chart_download_button

   d. **Drift Attribution** (st.expander "Attribution Breakdown"):
      - If drift timeseries has attribution columns (attr_fee_delta, attr_slippage_delta, etc.):
        - Show latest row attribution as st.columns or small table
        - Columns: Fees, Slippage, Timing, Data Revision, Sizing, Regime, Unexplained
        - Format as dollar values
      - If no attribution data: st.info("Attribution not yet computed")

   e. **Drift Summary Table** (expandable):
      - If show_history or always shown:
        - Load full drift_summary
        - st.dataframe with all columns, use_container_width=True
      - Else: hidden behind the checkbox

   f. **Refresh caption**

**Important**: v_drift_summary is a materialized view. Always show `last_metric_date` to
indicate data freshness. If the view is empty, show st.info("No drift metrics computed yet.
Run the drift monitor first.")

**Error handling**: try/except pattern matching existing pages.

      - st.subheader("Executor Run Log")
      - Load run_log with run_limit
