**Change 1: Add `_load_macro_regime_with_staleness_check()` helper function**

Add this function in the "Internal helpers" section (after `_forward_fill_labels`, around line 120). This function:
- Queries `SELECT regime_key, date FROM public.cmc_macro_regimes WHERE profile = :profile ORDER BY date DESC LIMIT 1`
- Catches `Exception` (including UndefinedTable when Phase 67 is not yet applied) and returns None
- If row is None (empty table), logs warning and sends Telegram alert, returns None
- Computes staleness as `(date.today() - regime_date).days`
- If staleness > `_L4_STALENESS_DAYS` (7), logs warning, sends Telegram alert, returns None
- Otherwise, logs the regime key and returns it

Use the `_try_telegram_alert` pattern: wrap `send_critical_alert("regime_refresh", message)` in try/except so alerting failure never crashes the refresh.

```python
_L4_STALENESS_DAYS = 7  # Disable L4 if macro regime not updated in 7+ days

def _load_macro_regime_with_staleness_check(
    engine: Engine,
    profile: str = "default",
) -> Optional[str]:
    """
    Load the latest macro regime composite key from cmc_macro_regimes.

    Returns the regime_key string (e.g. 'Cutting-Expanding-RiskOn-Stable'),
    or None if the table doesn't exist, is empty, or data is stale.
    Sends Telegram alert when L4 is disabled.
    """
    from datetime import date as _date  # noqa: PLC0415

    def _notify(msg: str) -> None:
        try:
            from ta_lab2.notifications.telegram import send_critical_alert  # noqa: PLC0415
            send_critical_alert("regime_refresh", msg)
        except Exception:  # noqa: BLE001
            logger.debug("_notify: Telegram unavailable")

    try:
        with engine.connect() as conn:
            row = conn.execute(
                text(
                    "SELECT regime_key, date FROM public.cmc_macro_regimes "
                    "WHERE profile = :profile ORDER BY date DESC LIMIT 1"
                ),
                {"profile": profile},
            ).fetchone()
    except Exception as exc:  # noqa: BLE001
        logger.warning(
            "L4 load failed (cmc_macro_regimes may not exist): %s", exc
        )
        _notify(f"L4 macro regime offline: table query failed ({exc})")
        return None

    if row is None:
        logger.warning(
            "L4 disabled: cmc_macro_regimes is empty for profile=%s", profile
        )
        _notify("L4 macro regime offline: cmc_macro_regimes is empty")
        return None

    regime_key, regime_date = row.regime_key, row.date
    if isinstance(regime_date, str):
        from datetime import datetime as _dt  # noqa: PLC0415
        regime_date = _dt.strptime(regime_date, "%Y-%m-%d").date()

    staleness = (_date.today() - regime_date).days
    if staleness > _L4_STALENESS_DAYS:
        logger.warning(
            "L4 disabled: macro regime is %d days stale (threshold=%d)",
            staleness,
            _L4_STALENESS_DAYS,
        )
        _notify(
            f"L4 macro regime offline: last update {staleness}d ago "
            f"(threshold={_L4_STALENESS_DAYS}d)"
        )
        return None

    logger.info(
        "L4 macro regime: %s (date=%s, %d days old)",
        regime_key,
        regime_date,
        staleness,
    )
    return regime_key
```
