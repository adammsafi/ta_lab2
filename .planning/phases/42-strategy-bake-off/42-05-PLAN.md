Create `src/ta_lab2/scripts/analysis/generate_bakeoff_scorecard.py` that reads from DB and CSV files to generate the full scorecard:

1. **Data loading functions**:
   - `load_ic_ranking()`: Read `reports/bakeoff/feature_ic_ranking.csv`
   - `load_composite_scores()`: Read `reports/bakeoff/composite_scores.csv`
   - `load_sensitivity()`: Read `reports/bakeoff/sensitivity_analysis.csv`
   - `load_bakeoff_results(engine)`: Query strategy_bakeoff_results for full metrics
   - `load_fold_details(engine, strategy, asset_id, tf)`: Extract fold_metrics_json for equity curve reconstruction

2. **Chart generation functions** (using Plotly, save as PNG via `fig.write_image()`; if kaleido not available, save as HTML):
   - `generate_ic_decay_chart(ic_ranking_df, top_n=10)`: Bar chart of top-10 features by |IC| at horizon=1
   - `generate_strategy_comparison_chart(scores_df)`: Grouped bar chart comparing strategies across metrics (Sharpe, Max DD, PSR, Composite)
   - `generate_cost_sensitivity_chart(results_df, strategy_name)`: Line chart of Sharpe vs slippage level per strategy
   - `generate_fold_equity_curves(fold_details, strategy_name)`: Overlaid equity curves from each fold (if fold equity data available in fold_metrics_json)
   - `generate_sensitivity_heatmap(sensitivity_df)`: Heatmap of strategy rank under each weighting scheme
