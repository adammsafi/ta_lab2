  <name>Task 1: Run ExperimentRunner sweep for all experimental features</name>
  <files>reports/evaluation/experiment_results.csv</files>
  <action>
1. **Run the full experiment sweep.** Use the run_experiment.py CLI to score all experimental features:
   ```bash
   python -m ta_lab2.scripts.experiments.run_experiment \
       --all-experimental \
       --train-start 2018-01-01 \
       --train-end 2025-12-31 \
       --yes
   ```
   This runs each feature defined with `lifecycle: experimental` in features.yaml through ExperimentRunner, which:
   - Loads input data from specified tables
   - Evaluates the inline/dotpath expression
   - Computes Spearman IC against forward returns at all configured horizons (1,2,3,5,10,20,60)
   - Applies BH correction across all hypotheses
   - Writes results to cmc_feature_experiments via upsert

   NOTE: With ~130+ features x multiple assets x horizons, this may take considerable time. The ExperimentRunner uses per-feature transactions so it can be interrupted safely.

   If the full sweep is prohibitively slow (>4 hours), batch by tag:
   - Stage 1: canonical features (highest priority for dashboard)
   - Stage 2: AMA features
   - Stage 3: EMA crossovers and adaptive RSI

   Use `--feature <name>` to run individual features if batching by tag is not directly supported.

2. **Export results to CSV.** After the sweep (or first stage):
   ```python
   # Query cmc_feature_experiments and write to CSV
   import pandas as pd
   from sqlalchemy import create_engine, pool, text
   from ta_lab2.scripts.refresh_utils import resolve_db_url

   engine = create_engine(resolve_db_url(), poolclass=pool.NullPool)
   with engine.connect() as conn:
       df = pd.read_sql(text("SELECT * FROM cmc_feature_experiments ORDER BY feature_name, asset_id, tf, horizon"), conn)
   df.to_csv("reports/evaluation/experiment_results.csv", index=False)
   print(f"Exported {len(df)} experiment rows")
   ```

3. **Generate BH gate summary.** For each feature, summarize whether it passes the BH gate:
   ```python
   # Group by feature_name, check if any horizon passes BH at alpha=0.05
   summary = df.groupby("feature_name").agg(
       n_experiments=("ic", "count"),
       n_significant=("ic_p_value_bh", lambda x: (x < 0.05).sum()),
       mean_abs_ic=("ic", lambda x: x.abs().mean()),
       best_ic_ir=("ic_ir", "max"),
   ).reset_index()
   summary["bh_passes"] = summary["n_significant"] > 0
   summary = summary.sort_values("mean_abs_ic", ascending=False)
   summary.to_csv("reports/evaluation/bh_gate_results.csv", index=False)
   ```
