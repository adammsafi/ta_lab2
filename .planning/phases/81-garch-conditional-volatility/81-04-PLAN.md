Add a new `garch_var` function to `var_simulator.py`. This computes conditional VaR
using a GARCH volatility forecast, complementing the existing historical, parametric
(normal), and Cornish-Fisher VaR methods.

**New function:**
```python
def garch_var(
    sigma_forecast: float,
    confidence: float = 0.95,
    mu: float = 0.0,
    dist: str = "studentst",
    df: float = 6.0,
) -> float:
    """GARCH conditional VaR.

    Uses the GARCH-forecasted conditional volatility to compute VaR via
    the distribution quantile: VaR = -(mu + sigma * q_alpha).

    Args:
        sigma_forecast: Daily conditional vol from GARCH (decimal, e.g. 0.03 = 3%).
        confidence:     Confidence level (default 0.95 for 95% VaR).
        mu:             Expected daily return (default 0 for conservative estimate).
        dist:           Distribution -- 'normal' or 'studentst' (default).
        df:             Degrees of freedom for Student's t (default 6.0, typical for crypto).

    Returns:
        Negative float representing the loss quantile, consistent with
        existing VaR functions in this module.
    """
```

**Implementation:**
- If `dist == "normal"`: `q = norm.ppf(1 - confidence)`
- If `dist == "studentst"`: `q = student_t.ppf(1 - confidence, df=df)`
- Return `mu + q * sigma_forecast` (negative for loss, consistent with existing convention)
- Validate: sigma_forecast > 0, confidence in (0, 1)

**Add Student's t import** at the top:
```python
from scipy.stats import t as student_t
```
(norm is already imported)
