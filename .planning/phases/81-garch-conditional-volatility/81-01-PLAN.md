**garch_engine.py:** Create the core GARCH fitting and forecasting module. This is a pure computation module (no DB, no CLI). It provides:

1. `MODEL_SPECS` dict mapping model_type strings to arch_model kwargs:
   - `garch_1_1`: `dict(vol="GARCH", p=1, o=0, q=1)`
   - `gjr_garch_1_1`: `dict(vol="GARCH", p=1, o=1, q=1)` (o=1 activates GJR threshold)
   - `egarch_1_1`: `dict(vol="EGARCH", p=1, o=1, q=1)`
   - `figarch_1_d_1`: `dict(vol="FIGARCH", p=1, q=1)`

2. `FIGARCH_MIN_OBS = 200` constant (gate FIGARCH behind higher minimum; research recommends 200-250, using 200 to maximize asset coverage while maintaining convergence reliability)

3. `DEFAULT_MIN_OBS = 126` constant

4. `@dataclass GARCHResult` with fields: model_type (str), converged (bool), convergence_flag (int | None), aic (float | None), bic (float | None), loglikelihood (float | None), ljung_box_pvalue (float | None), n_obs (int), h1_vol (float | None), h5_vol (float | None), error_msg (str | None), conditional_volatility (np.ndarray | None)

5. `fit_single_variant(returns_decimal: np.ndarray, model_type: str) -> GARCHResult`:
   - Scale returns by 100 before fitting (convergence aid -- research anti-pattern #1)
   - Use `dist="StudentsT"` for all variants (fat tails in crypto)
   - Use `rescale=True` as additional convergence aid
   - `maxiter=500` in fit options
   - `disp="off"` to suppress optimizer output
   - Gate FIGARCH behind `FIGARCH_MIN_OBS` check
   - If exception or convergence_flag != 0, return GARCHResult with converged=False and error_msg
   - Compute Ljung-Box p-value on standardized residuals (min across 10 lags) using `statsmodels.stats.diagnostic.acorr_ljungbox`
   - Extract conditional_volatility and unscale (divide by 100) back to decimal space
   - Return GARCHResult with all fields populated

6. `generate_forecasts(fit_result, model_type: str, horizon: int = 5) -> dict`:
   - For GARCH(1,1) and GJR-GARCH: use `method='analytic'` for all horizons
   - For EGARCH and FIGARCH: use `method='simulation'` when horizon > 1 (research pitfall #1)
   - Use `simulations=500` for simulation method
   - Extract h.1 and h.5 variance from `forecasts.variance` DataFrame
   - Unscale: divide variance by 10000 (100^2), then sqrt to get vol in decimal space
   - Return dict with keys `h1_vol`, `h5_vol`, `method_h5`
   - Handle exceptions gracefully (return None values with error key)

7. `fit_all_variants(returns_decimal: np.ndarray, min_obs: int = DEFAULT_MIN_OBS) -> dict[str, GARCHResult]`:
   - Validate len(returns) >= min_obs
   - Iterate MODEL_SPECS, calling fit_single_variant for each
   - For each converged result, also call generate_forecasts and merge h1_vol/h5_vol into the GARCHResult
   - Return dict mapping model_type -> GARCHResult

8. `compute_ljung_box_pvalue(std_resid: np.ndarray, lags: int = 10) -> float`:
   - Internal helper used by fit_single_variant
   - Uses `acorr_ljungbox` from statsmodels, returns min p-value

Use `from __future__ import annotations` at top. Add proper docstrings. Use logging (logger = logging.getLogger(__name__)). Import arch lazily with try/except to allow the module to be imported even if arch is not installed (matching project pattern in vol_sizer.py with vectorbt).
