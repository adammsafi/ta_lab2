**Dimension labelers** (private methods or module-level functions, pure logic, threshold-driven):

1. `_label_monetary(row, thresholds) -> str | None`:
   - Primary: use `fed_regime_trajectory` column from Phase 66 if not NaN. Normalize to title case: Hiking/Holding/Cutting.
   - Fallback: if fed_regime_trajectory is NaN, compute from `dff` 90d change vs thresholds.
   - Return None if all inputs are NaN.

2. `_label_liquidity(row, thresholds) -> str | None`:
   - Read `net_liquidity_trend` (Text: "expanding"/"contracting"/NULL from Phase 66) and `net_liquidity_365d_zscore` (Float).
   - Z-score extremes override the trend label:
     - If `net_liquidity_365d_zscore` > `strongly_expanding_z` threshold -> Strongly_Expanding
     - If `net_liquidity_365d_zscore` < `strongly_contracting_z` threshold -> Strongly_Contracting
   - Base direction uses `net_liquidity_trend` text label from Phase 66 (already computed as 30d MA vs 150d MA):
     - If trend == "expanding" -> Expanding
     - If trend == "contracting" -> Contracting
   - If neither z-score extreme nor trend applies -> Neutral
   - Return None if both `net_liquidity_trend` is NaN/NULL and `net_liquidity_365d_zscore` is NaN.
   - Implementation:
     ```python
     def _label_liquidity(row, thresholds) -> str | None:
         trend = row.get("net_liquidity_trend")  # "expanding"/"contracting"/NULL
         zscore = row.get("net_liquidity_365d_zscore")

         if pd.isna(trend) and pd.isna(zscore):
             return None

         # Z-score extremes override trend label for "strongly" qualifiers
         if not pd.isna(zscore):
             if zscore > thresholds["strongly_expanding_z"]:
                 return "Strongly_Expanding"
             if zscore < thresholds["strongly_contracting_z"]:
                 return "Strongly_Contracting"

         # Use trend from Phase 66 (already computed as 30d MA vs 150d MA)
         if trend == "expanding":
             return "Expanding"
         elif trend == "contracting":
             return "Contracting"

         return "Neutral"
     ```

def _compute_macro_state(regime_key: str, rules: dict) -> str:
    # Check each state's prefix patterns in priority order
    for state in ("adverse", "cautious", "neutral", "constructive", "favorable"):
        for pattern in rules.get(state, []):
            if regime_key.startswith(pattern):
                return state
    return rules.get("_default", "neutral")
