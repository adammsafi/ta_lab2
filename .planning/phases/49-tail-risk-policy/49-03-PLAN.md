**Composite score (from research):**
```python
def _composite_score(metrics: dict, baseline_worst_5: float) -> float:
    """0.4*Sharpe + 0.3*Sortino + 0.2*(1+Calmar) + 0.1*(1-|worst_5/baseline|)"""
    sharpe = metrics.get("sharpe", 0)
    sortino_val = metrics.get("sortino", 0)
    calmar_val = metrics.get("calmar", 0)
    worst_5 = metrics.get("worst_5_day_mean", baseline_worst_5)
    tail_component = 1.0 - abs(worst_5 / (baseline_worst_5 or -0.01))
    return 0.4 * sharpe + 0.3 * sortino_val + 0.2 * (1 + calmar_val) + 0.1 * max(tail_component, 0)
```
