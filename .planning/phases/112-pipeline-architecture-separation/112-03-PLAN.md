**PIPELINE_NAME = "signals"**

**Stages (in order per CONTEXT.md -- macro gates FIRST, not after features):**
1. macro_gates (via `run_evaluate_macro_gates`) -- pre-flight gate on macro conditions
2. macro_alerts (via `run_macro_alerts`) -- transition detection + Telegram
3. signals (via `run_signal_refreshes`) -- all 7 signal types
4. signal_validation_gate (via `run_signal_validation_gate`) -- anomaly detection, rc=2 blocks
5. ic_staleness_check (via `run_ic_staleness_check_stage`) -- IC freshness

**signal_gate_blocked logic:**
After `run_signal_validation_gate`, check if returncode == 2. If so, set `signal_gate_blocked = True`. This flag is used to decide whether to trigger sync_signals_to_vm in the chain step. When gate is blocked, the chain step skips the sync (so the VM executor sees no new signals and does nothing). Print a clear message: "[GATE] Signal validation gate BLOCKED -- signals will NOT be synced to VM".

**Argparse args:**
- `--db-url`: Database URL
- `--dry-run`, `--verbose`, `--continue-on-error`
- `--no-signal-gate`: Skip signal_validation_gate stage
- `--no-ic-staleness`: Skip ic_staleness_check stage
- `--from-stage`: Start from a specific stage within Signals stages
- `--chain`: After completion (if not gate-blocked), trigger sync_signals_to_vm subprocess

```python
def _get_last_execution_ts(db_url: str) -> datetime | None:
    """Query MAX(completed_at) from pipeline_run_log WHERE pipeline_name='execution' AND status='complete'."""

def run_polling_loop(args, db_url: str) -> int:
    while True:
        last_signal_ts = _get_last_signal_ts(db_url)
        last_exec_ts = _get_last_execution_ts(db_url)
        if last_signal_ts and (last_exec_ts is None or last_signal_ts > last_exec_ts):
            print(f"[EXEC] Fresh signals detected (ts={last_signal_ts}) -- running")
            exit_code = run_single_pass(args, db_url)
            if exit_code != 0:
                from ta_lab2.notifications import telegram
                if telegram.is_configured():
                    telegram.send_alert("Executor Pipeline Failed",
                        f"Exit code {exit_code}. Last signal ts: {last_signal_ts}",
                        severity="warning")
        else:
            print(f"[EXEC] No new signals (last_signal={last_signal_ts}, last_exec={last_exec_ts}) -- sleeping {args.poll_interval}s")
        time.sleep(args.poll_interval)
```
