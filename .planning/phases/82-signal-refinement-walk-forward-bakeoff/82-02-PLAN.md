**Signal 1: ama_momentum_signal (momentum/trend-following archetype)**
```python
def ama_momentum_signal(
    df: pd.DataFrame,
    *,
    ama_cols: list[str] | None = None,  # AMA feature columns to combine
    weights: list[float] | None = None,  # IC-IR weights (normalized)
    holding_bars: int = 7,  # swing trading: ~1 week holding
    threshold: float = 0.0,  # composite value threshold for entry
    allow_shorts: bool = False,
) -> Tuple[pd.Series, pd.Series, Optional[pd.Series]]:
```
Logic:
- Default ama_cols: top-5 by IC-IR = ["TEMA_0fca19a1_ama", "KAMA_987fc105_ama", "HMA_514ffe35_ama", "TEMA_514ffe35_ama", "DEMA_0fca19a1_ama"]
- Default weights: normalized IC-IR = [1.433, 1.291, 1.271, 1.257, 1.376] / sum
- Compute weighted composite: sum(w_i * df[col_i]) for available cols (skip missing gracefully)
- Compute rolling z-score of composite (20-bar window) for normalization
- Entry (long): z-score > threshold (default 0). Exit: z-score < -threshold OR held for holding_bars
- For holding_bars exit: track bars since entry using shift() logic
- If allow_shorts: short entries when z-score < -threshold
- Return (entries, exits, None) -- no position sizing from signal

**Signal 2: ama_mean_reversion_signal (mean-reversion archetype)**
```python
def ama_mean_reversion_signal(
    df: pd.DataFrame,
    *,
    ama_col: str = "KAMA_de1106d5_ama",  # AMA to revert to
    price_col: str = "close",
    zscore_window: int = 20,
    entry_zscore: float = -1.5,   # enter long when price below AMA by 1.5 sigma
    exit_zscore: float = 0.0,     # exit when price returns to AMA
    holding_bars: int = 10,       # max hold 10 days
    allow_shorts: bool = False,
) -> Tuple[pd.Series, pd.Series, Optional[pd.Series]]:
```
Logic:
- Compute spread = df[price_col] - df[ama_col]
- Z-score of spread using rolling Std(spread, zscore_window)
- Long entry: z-score < entry_zscore (price significantly below AMA)
- Long exit: z-score > exit_zscore (price returned to AMA) OR holding_bars exceeded
- If allow_shorts: short entry when z-score > abs(entry_zscore)
- CRITICAL: Handle NaN from AMA column gracefully (fillna(method=None), skip rows)

**Signal 3: ama_regime_conditional_signal (regime-conditional archetype)**
```python
def ama_regime_conditional_signal(
    df: pd.DataFrame,
    *,
    trend_col: str = "DEMA_d47fe5cc_ama",  # AMA trend indicator
    filter_col: str = "adx_14",    # ADX trend strength filter (computed locally if missing)
    adx_threshold: float = 20.0,   # minimum ADX for trend trades
    holding_bars: int = 7,
    allow_shorts: bool = False,
) -> Tuple[pd.Series, pd.Series, Optional[pd.Series]]:
```
Logic:
- If filter_col not in df.columns, compute ADX locally from OHLC (14-period)
- Compute trend direction: Sign(df[trend_col].diff())
- Condition on ADX: only enter when ADX > adx_threshold (trending market)
- Long entry: trend direction positive AND ADX > threshold
- Long exit: trend direction reverses OR ADX drops below threshold OR holding_bars exceeded
- If allow_shorts: short when trend negative AND ADX > threshold

**Common patterns for all three:**
- Use `pd.Series(False, index=df.index)` for initial entry/exit arrays
- Handle the holding_bars exit using a cumulative bars-since-entry counter
- fillna(False) on entry/exit series before returning
- Add module docstring explaining the three archetypes and that columns are pre-loaded
- Add `__all__` export list
