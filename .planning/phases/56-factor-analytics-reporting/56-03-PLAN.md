  <name>Task 1: Create quintile.py library module</name>
  <files>src/ta_lab2/analysis/quintile.py</files>
  <action>
    Create `src/ta_lab2/analysis/quintile.py` with two public functions:

    1. `compute_quintile_returns(features_df, factor_col, forward_horizon=1, close_col='close', min_assets_per_ts=5)`:
       - Input: DataFrame with columns including `ts`, `id`, `tf`, the `factor_col`, and `close_col`
       - At each `ts`: rank all assets by `factor_col` into 5 quintiles using `pd.qcut(x.rank(method='first'), 5, labels=[1,2,3,4,5])`
       - Filter timestamps where `COUNT(DISTINCT id) >= min_assets_per_ts` (default 5) -- quintile with fewer assets is meaningless
       - Compute forward returns per asset: `groupby('id')[close_col].transform(lambda x: x.shift(-forward_horizon) / x - 1.0)`
       - Average forward return per `(ts, quintile)` via `groupby(['ts', 'quintile'])['fwd_ret'].mean().unstack('quintile')`
       - Cumulative return per quintile: `(1 + quintile_returns).cumprod()`
       - Return tuple: `(cumulative_df, long_short_spread)` where `long_short_spread = cumulative_df[5] - cumulative_df[1]` (Q5 minus Q1)
       - Handle edge cases: if a timestamp has too few non-NaN factor values, skip it (do not crash on qcut with insufficient data)

    2. `build_quintile_returns_chart(quintile_cum_returns, factor_col, horizon, long_short_spread=None)`:
       - Input: DataFrame from compute_quintile_returns (columns 1-5, index=ts) + optional spread Series
       - Create Plotly figure with 5 lines (Q1-Q5) using colors: red, orange, green, blue, purple
       - If long_short_spread provided, add dashed black line for "Q5-Q1 Spread"
       - Title: f"Quintile Returns: {factor_col} (horizon={horizon}d)"
       - X-axis: "Date", Y-axis: "Cumulative Return"
       - Return `go.Figure`

    Module docstring explaining cross-sectional quintile analysis. Import: pandas, numpy, plotly.graph_objects, logging.

    CRITICAL anti-pattern from RESEARCH.md: Do NOT compute quintiles on a single asset's time series. This is cross-sectional ranking across ALL assets at each timestamp.
