## Milestone Summary

**Key Decisions:**
- Subprocess timeouts tiered by operation weight (bars=2h, EMAs=1h, stats=1h, git=30s)
- Pipeline gate unconditional for stats FAIL (ignores --continue-on-error)
- Weekly digest NOT in --all (separate reporting concern)
- Ruff zero-violations before removing || true escape hatch
- mypy scoped to features/ and regimes/ only (non-blocking)
- mkdocs-material pinned <9.7 (Windows colorama crash)
- No alembic autogenerate (write revisions by hand, stamp-then-forward only)
- Baseline revision 25f2b3c90f65 as Alembic epoch

**Issues Resolved:**
- All 30+ subprocess.run() calls now have timeout= (STAT-04)
- 31 ruff violations fixed (3 syntax, 15 auto, 13 manual)
- 106 files reformatted
- mkdocs --strict passing with all nav links resolved
- 17 legacy SQL migrations cataloged

**Technical Debt Accepted:**
- 1 subprocess.run() in emas/old/ without timeout (file is .gitignored)
- --no-telegram flag missing from DAILY_REFRESH.md flags table (code wiring correct)
