### Config Consistency

macro_regime_config.yaml thresholds are read exclusively by MacroRegimeClassifier via load_macro_regime_config(). MacroGateEvaluator has hardcoded thresholds (VIX > 30 for REDUCE, HY OAS z-score > 1.5/2.5, event windows in _EVENT_GATE_DEFAULTS). These are intentionally separate systems:

- Regime classifier thresholds: classify daily macro state, stored in cmc_macro_regimes
- Gate evaluator thresholds: block/reduce live orders, read from fred.fred_macro_features directly

VIX risk-off threshold in macro_regime_config.yaml (25.0 default) differs from VIX gate REDUCE threshold in macro_gate_evaluator.py (30.0). Intentional: regime classifier labels the environment conservatively; gates protect live orders at higher urgency bar.
